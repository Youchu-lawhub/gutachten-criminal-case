## §一 双权威源说明

本模板综合以下两大要素：

1. **刑法教义学方法论**：以阶层犯罪论体系为骨架，适配《刑法》条文与司法实践。
2. **鉴定式案例研习方法**：采用假设—定义—涵摄—结论四步法，确保法律推理的逻辑严密性与可追溯性。

## §三 9级标题精确矩阵

| 级别 | Markdown | 字体 | 字号 | 加粗 | 示例（刑法语境） |
|------|----------|------|------|------|------------------|
| 报告标题 | `#` | 黑体 | 小三(15pt) | 是 | `# 张三盗窃案鉴定式刑法案例分析报告` |
| 部分层 | `##` | 黑体 | 小三(15pt) | 是 | `## 第一部分 甲的刑事责任` |
| 一级·罪名 | `###` | 黑体 | 小三(15pt) | 是 | `### 甲持刀刺伤乙可能构成刑法第234条故意伤害罪 ✓` |
| 二级·大纲/正文 | `####` | 黑体 | 四号(14pt) | 是 | `#### （一）大纲` |
| 三级·阶层 | `#####` | 黑体 | 小四(12pt) | 是 | `##### 1. 构成要件符合性` |
| 四级·要件 | `######` | 宋体 | 小四(12pt) | 加粗 | `###### (1) 客观构成要件` |
| 五级·子要件 | `#######` | 宋体 | 小四(12pt) | 否 | `####### ① 因果关系与客观归责` |
| 六~七级 | 加粗段头 | 宋体 | 小四(12pt) | 加粗段头 | **A.** 条件公式 / **a.** 合法替代行为 |
| 八~九级 | 加粗段头 | 宋体 | 小四(12pt) | 加粗段头 | **(a)** 具体展开 / **(aa)** 极端细分 |

## §五 风格硬规则

1. **法条简称格式**：`刑法第XXX条第X款第X项`（使用中文数字）
2. **校验尾标**：法条引用块末尾必附三选一标签：
   - `[校验YYYY-MM-DD, 现行有效]`
   - `[⚠ 校验失败]`
   - `[⚠ 语义差异]`
3. **脚注**：非硬性覆盖率要求，但关键论断（学说观点、案例引用）应附脚注
4. **目录强制**：报告必须包含目录页（`# 目录`），位于报告标题之后
5. **部分层条件化**：仅当参与人 ≥ 2 时使用 `## 第X部分` 层级
6. **9级深度差异**：核心争议点可展开至7-9级，普通要件3-5级即可
7. **结论标记**：每一检验步骤末尾用 ✓ 或 ✗ 标注结论方向
8. **鉴定句式**：罪名标题必须使用"可能构成"句式

## §六 大纲与正文关系

| 维度 | 大纲 | 正文 |
|------|------|------|
| 功能 | 树状预览，展示检验路径 | 逐级展开论证 |
| 篇幅 | 30-80行 | 视争议密度而定 |
| 深度 | 至第五~七级 | 至第五~九级 |
| 论证 | 仅列标题+关键词 | 完整四步法论证 |
| 结论 | 在标题末标注 ✓/✗ | 在段末给出文字结论 |

### 篇幅指引

- **核心争议要件**：正文 ≥ 500字，含学说争议展开
- **普通要件**：正文 ≥ 250字，简明涵摄即可
- **无争议要件**：一句话带过或合并处理

## §七 单要素示范片段

### 示范一：鉴定式涵摄——故意杀人罪的"着手实行"

```
####### ② 着手实行

　　A是否已着手实行故意杀人？

　　着手实行是指行为人基于犯罪决意，开始实施与构
成要件密切相连的行为，依行为人的犯罪计划，其行为已对法益构成紧迫、直接的
危险（主客观混合说）。^[参见参考文献①第 298 页]

　　本案中，A持刀朝B胸部刺去。该行为已非预备行为（如购买凶器、跟踪被害
人），而是直接指向杀人罪构成要件结果（他人死亡）的实现行为。依A的犯罪计划，
持刀刺击已使B的生命法益面临紧迫危险。

　　因此，A已着手实行故意杀人。✓
```

### 示范二：学说争议展开——打击错误

```
####### ③ 打击错误

　　A瞄准B射击却击中C，打击错误是否排除A对C死亡的故意？

　　关于打击错误是否排除故意，学说上存在分歧：

　　**A. 法定符合说**认为，只要行为人的认识与实
际发生的事实属于同一构成要件范围，即不影响故意的成立。本案中，A瞄准B开枪
却误中C，B与C均为"他人"，属于故意杀人罪的同一构成要件，故A对C的死亡具
有故意。

　　**a. 具体符合说**则认为，故意要求行为人的认
识与实际发生的具体事实相一致。A主观上认识的对象是B而非C，二者并非同一具
体对象，故A对C不具有杀人故意，仅成立对C的过失致人死亡罪；同时对B成立故
意杀人未遂。

　　本文采法定符合说。理由在于：故意杀人罪保护的是"人"这一抽象法益主体，
而非特定个体。A具有杀人故意且客观上导致了"他人"死亡，在规范评价上应认定
故意杀人既遂。^[参见参考文献②"打击错误"相关章节]

　　因此，A对C的死亡成立故意杀人罪（既遂）。✓
```
