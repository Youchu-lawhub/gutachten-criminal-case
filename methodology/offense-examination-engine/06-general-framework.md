# 通用检视程式骨架库

> 本文件提供九种常见犯罪类型的鉴定式检视模板骨架。每种类型均标注《刑法》条文依据。
>
> 框架库基于中华人民共和国现行《刑法》（2023 修正）。中国法未明文规定的事项（如违法性认识、容许构成要件错误、共犯从属性程度等），相关展开属学理解释。

---

## 使用说明

1. **模板性质**: 下列骨架为起始参照点，非穷尽方案。具体案件须根据罪名条文、司法解释及案例进行调整。
2. **MCP 验证义务**: 任何模板中的条文号、构成要件要素均须经 MCP 工具核实后方可写入最终报告。
3. **未覆盖类型**: 本文件未涵盖的犯罪类型（如危险犯、单位犯罪等）须从教义学基本原理出发自行建构。
4. **引用格式**: 引文统一使用"《刑法》第N条"的中文格式。

## 1. 故意作为既遂犯

适用条件：行为人以积极作为方式实施故意犯罪且达到既遂状态。

```
I. 构成要件符合性

  A. 客观构成要件

    1. 行为主体
       — 自然人（单位犯罪另论）
       — 身份犯须具备特殊身份（如国家工作人员）

    2. 构成要件行为 / 实行行为
       — 意志支配的身体动作
       — 实行行为性：事前视角判断的类型化危险创设
       — 作为 / 不作为区分（不作为转入第4类模板）

    3. 行为对象
       — 仅当条文明确要求时检验（如"公私财物"、"他人"）

    4. 构成要件结果
       — 条文所描述的法益侵害结果

    5. 因果关系与客观归责
       (a) 条件说事实归因
       (b) 规范归责:
           — 规范保护目的
           — 义务违反关联
           — 被害人自我答责（被害人自愿且明知地承担风险时，结果归责于被害人）
           — 第三人介入
           — 因果流程偏离

  B. 主观构成要件

    1. 故意
       (a) 认识要素: 对全部客观构成要件事实的认识
       (b) 意志要素:
           — 直接故意 / 希望
           — 间接故意 / 放任

    2. 主观超过要素
       — 如非法占有目的、牟利目的、特定动机等

  C. 故意既遂归责对应检验
     — 仅当主观想象 ≠ 客观现实时进入检验
     — 打击错误 / 客体错误 / 因果流程偏离

II. 违法性 — 消极检验（推定违法，逐一排除正当化事由）

  A. 正当防卫（《刑法》第20条）
     — 防卫情状: 正在进行的不法侵害
     — 防卫行为: 必要性 + 相当性
     — 防卫意思

  B. 紧急避险（《刑法》第21条）
     — 避险情状: 正在发生的危险
     — 避险行为: 不得已 + 限度（所保护利益 > 所损害利益）
     — 避险意思

  C. 被害人承诺
     — 法益可处分性 + 承诺能力 + 自愿性 + 事前承诺 + 行为人认知

  D. 法令行为 / 其他正当化事由
     — 依法律或职务的行为（如警察执行公务、医生紧急救治）——中国《刑法》对此无总则明文规定，属正当化事由的学理类型，依各部门法判断
     — 正当业务行为

III. 有责性 — 消极检验

  A. 责任能力（《刑法》第17条、第17条之一、第18条、第19条）
     — 年龄（《刑法》第17条）：未满14周岁完全无责任能力；已满14未满16仅对8种重罪负责；已满75周岁可从轻或减轻（第17条之一）
     — 精神状态（《刑法》第18条）：精神病人 / 限制责任能力人

  B. 违法性认识可能性
     — 不可避免的禁止错误 → 学理通说采责任说，排除有责性
     — 可避免的禁止错误 → 可从轻处罚

  C. 期待可能性
     — 学理上的有责性减免事由

  D. 责任故意（容许构成要件错误情形）
     — 假想防卫：
       学理上有严格责任说/限制责任说/法律效果转用的限制责任说之争；
       本框架默认采"法律效果转用的限制责任说"（按事实错误的法律效果处理，阻却故意责任）

IV. 其他要件

  A. 客观处罚条件
     — 不属于不法/有责，但为可罚性前提（如部分罪名规定的"数额较大"门槛）

  B. 刑罚解除事由
     — 若涉及亲属间盗窃，参《最高人民法院关于审理盗窃案件具体应用法律若干问题的解释》对偷拿自家或近亲属财物的特别处理（一般不按犯罪处理，确需追诉的应酌情从宽）
```

## 2. 未遂犯（《刑法》第23条）

适用条件：行为人已着手实行但犯罪未达既遂。

```
前置检验:
  (1) 犯罪未既遂: 结果未发生或要件不完整
  (2) 未遂可罚性:
      — 依《刑法》第23条第1款，对未遂犯可以比照既遂犯从轻或者减轻处罚
      — 中国法对故意犯罪原则上承认未遂；举动犯（如脱逃罪）等特殊罪名理论上不能成立未遂——具体由分则规定与司法解释确定

I. 构成要件符合性

  A. 犯罪决意 [先于客观检验]
     — 行为人主观上须覆盖全部客观构成要件要素的故意
     — 包括主观超过要素（如非法占有目的）

  B. 着手实行
     — 判断基础: 主客观混合说
     — 具体标准: 依行为人对事实的设想,
       行为已对受保护法益构成紧迫危险
     — 间接正犯着手: 脱手理论（间接正犯完成利用行为、被利用者进入实行阶段时即为着手）

II. 违法性
   — 同既遂犯框架
   — 争议: 正当化事由的事实前提在想象中是否充分（假想正当防卫 + 未遂）

III. 有责性
   — 同既遂犯框架

后置检验:
  (1) 不能犯（学理概念）
     — 注：中国《刑法》第23条仅对未遂作一般规定，未对不能犯设独立条款（中国《刑法》第23条无第2款）。学理上多数说将不能犯分为：
       · 绝对不能犯（对象绝对不能/手段绝对不能且根本无危险）→ 不可罚
       · 相对不能犯（具体情境下虽未得逞但仍有抽象危险）→ 仍按未遂处罚
       · 迷信犯 → 不可罚
     — 本框架默认采此学说立场

  (2) 中止犯（《刑法》第24条）
     — 未实行终了: 自动放弃
     — 已实行终了: 自动有效防止结果发生
     — 自动性: "能达目的而不欲"vs"欲达目的而不能"
     — 法律后果（《刑法》第24条）：没有造成损害的，**应当**免除处罚；造成损害的，**应当**减轻处罚
```

## 3. 过失犯 — 二阶过失论

适用条件：行为人因违反注意义务过失地造成法益侵害结果。

```
I. 构成要件符合性

  A. 客观构成要件基础要素
     — 行为主体 / 行为 / 行为对象 / 结果 / 条件说因果关系

  B. 客观注意义务违反 — 理性人标准
     (a) 结果预见义务:
         以理性谨慎人在行为时能否预见结果
     (b) 结果避免义务:
         行为人应采取何种措施避免结果
     (c) 客观可预见性:
         结果类型及基本因果流程在客观上可预见
     (d) 义务违反关联 / 风险实现:
         — 若行为人遵守注意义务，结果"几乎确定"不会发生
         — 合义务替代行为检验
         — 规范保护目的

II. 违法性
   — 同故意犯框架（正当防卫、紧急避险、被害人承诺等）
   — 信赖原则作为注意义务限缩理由（合理信赖他人会遵守规则时，不对他人违规造成的结果负责）
   — 允许的风险

III. 有责性

  A. 责任能力（《刑法》第17条、第17条之一、第18条、第19条）

  B. 违法性认识可能性

  C. 期待可能性

  D. 主观注意义务违反 / 责任过失
     — 行为人个人标准
     (a) 主观可预见性: 以行为人个人知识、经验、能力能否预见
     (b) 个人履行义务的可能性: 行为人个人是否有能力避免结果

IV. 其他要件
   — 客观处罚条件（如有）
   — 部分"告诉才处理"罪名（如《刑法》第246条侮辱诽谤、第257条暴力干涉婚姻自由等，依分则具体规定）
```

## 4. 不真正不作为犯

适用条件：行为人负有防止结果发生的保证人义务，但未采取行动。

```
前置检验:
  (1) 不作为性确认:
      — 重点考察: 行为对法益损害的核心贡献在"不行动"
  (2) 等置条款:
      — 不作为在规范评价上须"相当于"以作为方式实施

I. 构成要件符合性

  A. 客观构成要件
     (1) 构成要件结果发生
     (2) 命令之特定行为: 行为人可能采取的防止结果的具体作为
     (3) 作为可能性:
         行为人在当时情况下身体上能够实施命令行为
     (4) 假设因果关系:
         若行为人实施命令行为，结果"几乎确定"不会发生
     (5) 保证人地位:
         a. 保护型保证人:
            — 自然紧密关系 / 共同体关系 / 自愿承担保护义务 / 职务上的保护义务
         b. 监督型保证人:
            — 对危险源的监督义务 / 先行行为 / 对物的管领义务
     (6) 等置性:
         不作为在不法内涵上相当于作为

  B. 主观构成要件
     — 故意须覆盖上述全部客观要素
     — 含对保证人地位基础事实的认识

II. 违法性
   — 标准正当化事由
   — 特别考量: 义务冲突
     — 同等价值义务冲突时: 履行其一即合法
     — 不等价值义务冲突: 须履行较高价值义务

III. 有责性
   — 标准有责性框架
   — 特别考量: 诫命错误
     — 对保证人义务存在的错误认识 → 按禁止错误规则处理（不可避免则排除责任，可避免则减轻）
```

## 5. 共同正犯

适用条件：二人以上基于共同犯罪计划共同实施犯罪。

> 注：《刑法》第25条规定共同犯罪定义（"二人以上共同故意犯罪"），未明文区分"共同正犯/间接正犯/教唆犯/帮助犯"。《刑法》明文规定的参与人类型为主犯（第26条）/从犯（第27条）/胁从犯（第28条）/教唆犯（第29条）。本检视框架借助教义学方法精细定性，最终归责仍须回到四类参与人体系。

```
I. 构成要件符合性

  A. 客观前提
     — 构成要件结果发生
     — 身份要件满足（身份犯中须至少一人具备身份）

  B. 共同正犯特殊要件
     (1) 共同犯罪计划:
         — 明示或默示的意思联络
         — 各人对犯罪整体有概括认识
     (2) 共同实施:
         — 各参与人在实行阶段的功能性贡献
     (3) 功能支配 / 重要贡献:
         — 各人贡献对犯罪成功具有本质意义
         → "部分实行全部责任"原则

  C. 标准客观 + 主观构成要件
     — 通过相互归责: 他人实行行为视为自己的行为
     — 主观上: 共同故意覆盖犯罪整体
     — 注：中国《刑法》第25条第2款明文排除"二人以上共同过失犯罪"，故学理上的"过失共同正犯"在中国法上原则上不成立

II. 违法性 — 个别判断

III. 有责性 — 个别判断
```

## 6. 间接正犯

适用条件：利用他人作为工具实施犯罪。法律后果归入《刑法》第 25 条共同犯罪规范或按单独正犯处罚。

```
I. 构成要件符合性

  A. 客观构成要件
     (1) 构成要件结果发生
     (2) 间接正犯特殊要件:
         a. 前台人行为欠缺某犯罪成立要素
         b. 后台人的犯罪事实支配:
            — 强制支配: 前台人受强迫
            — 错误支配: 前台人陷入错误
            — 组织支配: 权力机构中的命令链
            — 能力欠缺之利用: 前台人无责任能力
         c. 支配意思: 后台人认识并意图支配犯罪过程

  B. 主观构成要件
     — 故意: 对通过前台人实现构成要件的全部事实有认识与意志

II. 违法性 — 就后台人单独判断

III. 有责性 — 就后台人单独判断
```

## 7. 教唆犯（《刑法》第29条）

适用条件：故意唆使他人实施犯罪。

> 注：《刑法》第29条规定教唆犯——"教唆他人犯罪的，应当按照他在共同犯罪中所起的作用处罚。教唆不满十八周岁的人犯罪的，应当从重处罚。如果被教唆的人没有犯被教唆的罪，对于教唆犯，可以从轻或者减轻处罚。"第29条第2款规定了"独立教唆"条款，教唆犯的可罚性不以正犯既遂为前提。

```
I. 构成要件符合性

  A. 客观构成要件
     (1) 正犯不法行为存在:
         — 至少达到未遂阶段（限制从属性说：正犯行为须具备违法性，但不要求有责性）
     (2) 教唆行为:
         — 引起犯意
         — 须使原无犯意之人产生犯意（已决意者不可教唆）
     (3) 因果关系:
         — 教唆行为与正犯犯意之间的因果联系

  B. 主观构成要件 — 双重故意
     (1) 教唆故意: 引起他人犯意的故意
     (2) 对正犯行为及结果的故意: 认识到正犯将实施特定犯罪

II. 违法性 — 就教唆犯自身判断

III. 有责性 — 就教唆犯自身判断

IV. 量刑特别规则
   — 第29条第1款：按其在共同犯罪中所起的作用处罚（起主要作用为主犯，起次要/辅助作用为从犯）
   — 第29条第2款：教唆不满18周岁的人犯罪——应当从重处罚
   — 第29条第3款：被教唆的人没有犯被教唆的罪——可以从轻或者减轻处罚（独立教唆）
```

## 8. 从犯（含学理上的"帮助犯"，《刑法》第27条）

适用条件：在共同犯罪中起次要作用或辅助作用。

> 注：中国《刑法》第27条规定的从犯包括两类——"次要作用"（在共同实行中起次要作用）与"辅助作用"（提供工具、传递信息等不直接实行的帮助），后者大致对应学理上的"帮助犯"。第27条法律后果："对于从犯，应当从轻、减轻处罚或者免除处罚。"——三档**应当**减免，无裁量空间。

```
I. 构成要件符合性

  A. 客观构成要件
     (1) 正犯不法行为存在:
         — 至少达到未遂阶段（限制从属性说：正犯行为须具备违法性，但不要求有责性）
     (2) 帮助行为:
         — 物理帮助: 提供工具、信息、场所等
         — 心理帮助: 强化犯意、提供技术咨询
     (3) 因果贡献:
         — 帮助行为须对正犯行为有所促进（降低门槛即可，无需条件关系）

  B. 主观构成要件 — 双重故意
     (1) 帮助故意: 认识到自己在帮助他人犯罪
     (2) 对正犯行为及结果的故意: 认识到正犯行为的主要特征

II. 违法性 — 就从犯自身判断
   — 特殊问题: 中立帮助行为的可罚性限制（学理）

III. 有责性 — 就从犯自身判断
   — 量刑（《刑法》第27条）：对从犯，**应当**从轻、减轻处罚或者免除处罚
```

## 9. 结果加重犯

适用条件：实施基本犯罪后发生更严重结果，法律规定加重刑罚。
典型罪名：故意伤害致死（《刑法》第234条）、抢劫致人重伤/死亡（《刑法》第263条）、强奸致人重伤/死亡（《刑法》第236条）等。

```
前置检验: 基本犯罪完整成立
  — 须完成基本犯罪的全部构成要件符合性 + 违法性 + 有责性检验

I. 加重构成要件

  A. 加重结果发生
     — 与基本犯罪结果不同的、更严重的法益侵害

  B. 基本犯罪与加重结果的因果关系
     — 条件说: 若无基本犯罪行为，加重结果不会发生

  C. 直接性要件:
     — 加重结果须为基本犯罪行为所蕴含的特有危险的直接实现
     — 非偶然的、外部介入导致的结果

  D. 主观要件: 对加重结果至少有过失
     — 类型一: 故意基本犯 + 过失加重结果（最常见，如故意伤害致死）
     — 类型二: 故意基本犯 + 故意加重结果（如抢劫杀人情形）
     — 类型三: 过失基本犯 + 过失加重结果（较罕见）

II. 违法性 — 同基本犯框架

III. 有责性 — 同基本犯框架
   — 特别注意: 对加重结果的主观过失须满足有责性层面的主观注意义务违反
```
