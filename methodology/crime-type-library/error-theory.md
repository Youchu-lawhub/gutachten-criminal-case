# 认识错误

> **体系定位**：认识错误理论的核心在于确定错误在三阶层犯罪论体系中的**处理位置**。不同类型的错误在不同阶层产生不同的法律效果。本文件按错误的体系处理位置进行分类。注意：《刑法》第 16 条为"不可抗力和意外事件"，不是构成要件错误条款。

---

## 总论：认识错误的体系分类

### 分类框架

```
认识错误体系
│
├── 第一类：排除故意的事实错误
│   └── 处理位置：构成要件符合性 → 主观构成要件
│   └── 效果：排除故意，可能成立过失犯
│
├── 第二类：影响归责对应的错误
│   └── 处理位置：构成要件符合性 → 归责对应检验
│   └── 效果：可能排除既遂故意归责
│
├── 第三类：影响有责性的错误
│   ├── 容许构成要件错误 → 有责性层
│   └── 禁止错误 → 有责性层
│   └── 效果：可能排除或减轻责任
│
└── 特殊情形：假想防卫过当
    └── 处理位置：综合处理
```

### 传统分类与体系分类的对照

| 传统分类 | 体系处理位置 |
|----------|-------------|
| 事实错误 | 主观构成要件 |
| 打击错误 | 归责对应/主观构成要件（争议） |
| 因果流程偏离 | 归责对应检验 |
| 容许构成要件错误 | 有责性 |
| 禁止错误 | 有责性 |

## 第一类：排除故意的事实错误

> **处理位置**：I. 构成要件符合性 → B. 主观构成要件 → 1. 故意

### 基本规则

```
规则：对客观构成要件要素事实的不认识，排除故意。
效果：故意犯不成立 → 检验过失犯是否成立

例：甲将他人的雨伞误认为自己的而取走
  → 甲不知"他人性"这一客观构成要件事实
  → 排除盗窃故意
  → 检验是否成立过失犯（盗窃罪无过失形态 → 不成立犯罪）
```

### 一、具体事实错误

#### 1. 对象错误（error in persona / error in objecto）

##### 概念

行为人攻击了预定对象，但对该对象的身份/性质产生了错误认识。

##### 处理规则

```
基本规则：对象错误原则上不排除故意

理由：
- 行为人意图攻击的对象 = 实际攻击的对象（物理上同一）
- 仅对对象的"附属属性"有错误认识
- 构成要件仅要求认识到"人"/"他人财物"等类别特征
- 对具体是"哪个人"/"谁的财物"的认识不是故意内容

例：甲意图杀A，黑暗中将B误认为A并开枪射杀B
  → 甲故意杀"人"→ B也是"人"→ 故意仍存在
  → 对B成立故意杀人既遂

例外——对象错误导致构成要件类别变更：
  甲意图杀人，但误将稻草人当作人射击
  → 对象并非"人"→ 故意杀人未遂（对象不能犯）
```

##### 涉及身份犯的特殊问题

| 情形 | 处理 |
|------|------|
| 意图杀普通人，实际杀死国家工作人员 | 不影响故意杀人的成立 |
| 以为对象有特殊身份实际没有 | 可能影响（如贿赂罪——非国家工作人员则构成要件不符） |

#### 2. 打击错误（aberratio ictus）

##### 概念

行为人意图攻击对象A，但因行为偏差（瞄准不准、风向改变等），实际击中了对象B。

##### 学说对立

| 学说 | 主张 | 论证 | 结论 |
|------|------|------|------|
| **法定符合说** | 同一构成要件范围内的对象在规范上等价 | 刑法保护"人"的生命，不保护"特定人"的生命 | 对B成立故意杀人既遂 |
| **具体符合说**【本程式库采】 | 故意须指向具体对象 | 行为人的"攻击意思"指向A而非B | 对A：故意杀人未遂；对B：过失致人死亡；两者想象竞合 |

##### 两说比较

```
案例：甲意图杀A，开枪射击A，子弹偏离击中旁边的B致死

法定符合说：
  → 甲故意杀"人"
  → B也是"人"
  → 对B成立故意杀人既遂
  → 对A可能附带成立故意杀人未遂

具体符合说：
  → 甲的杀人故意指向A（具体对象）
  → A未被击中 → 对A成立故意杀人未遂
  → B被击中但甲无杀B的故意 → 对B检验过失致人死亡
  → 想象竞合：故意杀人未遂 + 过失致人死亡 → 从一重处断
```

##### 争议焦点与本程式库立场

```
本程式库立场：原则采具体符合说

理由：
1. 故意的本质是对"具体犯罪事实"的认识与意欲
2. 行为人未认识到"攻击B"这一事实
3. 法定符合说过度抽象化故意内容
4. 具体符合说更符合故意的心理学基础

但标注法定符合说的结论差异，供使用者选择。

实务注意：
- 中国刑法实务倾向法定符合说（对实际被害人成立故意既遂）
- 鉴定式答案中应展示两说差异并明确表态
```

##### 打击错误 vs 对象错误的区分

| 区分标准 | 对象错误 | 打击错误 |
|----------|----------|----------|
| 攻击对象 | 攻击的就是面前这个对象（只是认错了身份） | 意图攻击A但实际击中B |
| 物理偏差 | 无物理偏差 | 有物理偏差/行为偏差 |
| 错误来源 | 认知层面的错误 | 执行层面的偏差 |

### 二、抽象事实错误

#### 概念

行为人认识的事实与客观事实分属不同构成要件（跨构成要件的错误）。

#### 类型与处理

##### 类型 1：有构成要件重叠

```
情形：行为人主观上构成罪名A的故意，客观上实现了罪名B的构成要件
      且A与B之间存在构成要件的"交集"

处理规则：
  在"重叠部分"范围内成立故意犯

例：甲以为攻击的是"人"（意图故意杀人），实际攻击对象为尸体
  → 故意杀人的构成要件要素"活的人"不满足
  → 无重叠 → 对"人"的未遂 + 对尸体无犯罪（毁坏尸体需独立检验）

例：甲以为窃取的是"他人之物"（盗窃），实际该物为遗忘物（侵占）
  → 盗窃与侵占在"非法取得他人财物"层面有重叠
  → 在重叠范围内成立侵占罪（轻罪）
```

##### 类型 2：无构成要件重叠

```
情形：行为人主观认知的犯罪类型与客观实现的犯罪类型完全不同

处理规则：
  对主观意图的犯罪：成立未遂（如可罚）
  对客观实现的犯罪：检验过失犯（如有规定）
  两者想象竞合

例：甲以为射击的是野猪（无犯意），实际射中猎人（过失致人死亡）
  → 故意面：无故意 → 不成立故意杀人
  → 过失面：是否违反注意义务？→ 可能成立过失致人死亡
```

## 第二类：影响归责对应的错误

> **处理位置**：I. 构成要件符合性 → C. 故意既遂归责对应检验

### 适用前提

- 主观上：行为人具有完整故意
- 客观上：构成要件结果已发生
- 问题：主观想象的因果流程与客观实际的因果流程不一致
- 目的：判断故意能否"对应"到实际结果上

### 一、因果流程偏离

#### 概念

行为人意图以特定方式实现结果，但实际的因果流程与其想象不同，结果仍然发生。

#### 处理标准——一般可预见性说（通说）

```
规则：
  若因果流程的偏离在"一般生活经验"范围内可预见
  → 不排除故意既遂归责
  → 以行为人想象的因果计划为基础判断偏离程度

判断步骤：
1. 确认行为人的因果想象（计划如何致果）
2. 确认实际因果流程
3. 比较两者偏差程度
4. 偏差是否超出一般可预见范围？
   - 否 → 故意既遂归责成立
   - 是 → 故意既遂归责中断 → 可能仅成立未遂 + 过失
```

#### 典型情形

| 情形 | 偏离程度 | 结论 |
|------|----------|------|
| 甲欲勒死乙后抛尸入河，乙实际溺死 | 可预见范围内 | 故意杀人既遂 |
| 甲射杀乙，乙未立即死亡，送医后感染死亡 | 可预见范围内 | 故意杀人既遂 |
| 甲推乙下桥欲溺死，乙撞桥墩死亡 | 可预见范围内 | 故意杀人既遂 |
| 甲伤乙后弃之，乙被救护车送医途中遭雷击死亡 | 超出可预见范围 | 杀人未遂 + 过失致死？ |

### 二、结果提前发生

```
问题：行为人计划分多步实施，但在预备阶段的行为已导致结果发生

例：甲计划先将乙灌醉（第一步），再将乙推入河中（第二步）
    结果：乙在灌醉后即因醉酒窒息死亡

分析：
- 第一步行为（灌醉）时：甲有何故意？
  → 若灌醉本身属于"着手实行"→ 故意杀人既遂（因果偏离在可预见范围）
  → 若灌醉仅为预备行为 → 尚未着手 → 不可直接归责为既遂
- 关键判断：第一步行为是否已构成"着手"？
```

### 三、结果推迟发生

```
问题：行为人以为已经得逞（第一行为后），实施了善后行为（第二行为），
      而结果实际由第二行为导致

经典案例：
  甲猛击乙头部（第一行为），以为乙已死亡
  → 甲为毁尸将乙抛入河中（第二行为）
  → 乙实际溺死于河中

处理方案：

方案一：整体评价说
  - 将第一行为和第二行为视为一个整体
  - 因果偏离在可预见范围内
  → 故意杀人既遂

方案二：分割评价说
  - 第一行为：故意杀人未遂（结果未发生）
  - 第二行为：过失致人死亡（以为已死实际未死）
  - 想象竞合

本程式库立场：
  采整体评价说——行为人自始具有完整杀人故意，实际因果偏离
  （"以为已死实际未死"）在一般可预见范围内。
```

### 四、修正风险理论（补充参考）

```
方法论：
1. 提取行为人认识的事实与客观发生事实的"重叠部分"
2. 构建一个新的"综合性危险描述"
3. 检验该综合描述是否构成"不被容许的风险"
4. 是 → 归责成立；否 → 归责不成立

适用场景：传统"一般可预见性"标准不明确时的辅助判断
```

## 第三类：影响有责性的错误

> **处理位置**：III. 有责性

### 一、容许构成要件错误

> 本程式库默认采"法律效果转用的限制责任说"。

#### 概念

行为人误以为存在正当化事由的**事实前提**，但该事实前提客观上不存在。

#### 典型情形

| 情形 | 说明 |
|------|------|
| 假想防卫 | 误以为对方在攻击自己，实施"防卫" |
| 假想紧急避险 | 误以为存在紧急危险，实施"避险" |
| 假想承诺 | 误以为被害人已同意 |

#### 处理步骤

```
Step 1: 客观判断——正当化事由的事实前提客观上不存在
  → 确认：客观上无现在的不法侵害/无紧急危险/无有效承诺
  → 结论：违法性未被排除（行为客观上违法）

Step 2: 主观判断——行为人主观上认为事实前提存在
  → 确认：行为人真诚相信正在受到攻击/存在危险/已获承诺
  → 进入有责性层处理

Step 3: 理论方案选择
```

#### 理论争议与处理方案

| 学说 | 处理方式 | 法律效果 |
|------|----------|----------|
| **严格责任说** | 类推禁止错误 | 故意犯成立；不可避免则阻却责任，可避免则从轻 |
| **限制责任说** | 类推事实错误（排除故意） | 排除故意犯处罚，可能成立过失犯 |
| **法律效果转用的限制责任说**【本程式库采】 | 直接适用禁止错误的法律效果但阻却故意犯的处罚 | 不成立故意犯；不可避免→不罚；可避免→可能过失犯 |

#### 本程式库采用的处理流程

```
容许构成要件错误的检验：

III. 有责性
  D. 容许构成要件错误
    1. 确认：行为人误认正当化事由的事实前提存在
    2. 效果（法律效果转用的限制责任说）：
       a. 排除故意犯的处罚
       b. 检验该错误是否"可避免"：
          - 不可避免 → 完全不罚
          - 可避免 → 检验过失犯是否成立
    3. 过失犯检验：
       - 是否存在该罪的过失犯处罚规定？
       - 行为人是否违反了注意义务？（是否有合理理由误认？）
```

### 二、禁止错误

> 学理通说采责任说处理。

#### 概念

行为人对自己行为的**违法性**（法律评价）产生错误认识——不知道自己的行为被法律禁止。

#### 与事实错误的根本区分

| 区分 | 事实错误 | 禁止错误 |
|------|----------|----------|
| 认识对象 | 对事实状况的认识 | 对法律评价的认识 |
| 错误内容 | "我不知道事实如何" | "我不知道这样做违法" |
| 效果 | 排除故意 | 不排除故意（仅影响有责性） |
| 体系位置 | 构成要件 | 有责性 |

#### 禁止错误的类型

```
1. 直接的禁止错误：
   - 不知存在禁止规范
   - 例：外国人不知中国禁止某种行为

2. 间接的禁止错误：
   - 知道存在禁止但误认为存在正当化事由
   - 子类型：
     a. 假想正当化事由：
        误以为法律上存在某种正当化事由（实际不存在该事由）
     b. 正当化事由范围错误：
        误认为正当化事由的适用范围比实际更宽
   - 注意：与容许构成要件错误的区分！
     - 容许构成要件错误 = 对事实前提的错误
     - 间接禁止错误 = 对法律规定本身的错误
```

#### 处理方案——责任说

```
禁止错误的法律效果：

1. 不可避免的禁止错误：
   → 阻却责任 → 不成立犯罪
   
2. 可避免的禁止错误：
   → 不阻却责任
   → 故意犯成立
   → 但可以减轻处罚

可避免性的判断标准：
  - 行为人是否有理由怀疑自己行为的合法性？
  - 行为人是否尽到了合理的法律咨询义务？
  - 是否利用了可获得的信息源？
  - 特殊领域行为人是否有更高的认知义务？
```

#### 禁止错误 vs 事实错误：边界案例

| 案例 | 分类 | 理由 |
|------|------|------|
| 不知"财物"包括电力 | 事实错误（通说） | 对构成要件要素的事实涵义不了解 |
| 不知"淫秽物品"的法律定义范围 | 禁止错误 | 对法律概念的评价性理解错误 |
| 误认对方年龄已满14周岁 | 事实错误 | 对描述性事实的认知错误 |
| 不知14周岁以下为性同意年龄界限 | 禁止错误 | 对法律规定本身不了解 |

## 第四类：特殊错误情形

### 一、假想犯罪

```
概念：行为人误以为自己的行为构成犯罪，但实际上不构成犯罪

与未遂犯的区分：
  - 假想犯罪："我以为违法但实际不违法"
    → 不可罚（无不法）
  - 不能犯未遂："我以为能够实现但实际不能"
    → 可罚（有犯罪决意 + 法秩序动摇印象）

例：
  甲以为通奸是犯罪而与他人发生性关系 → 假想犯罪，不可罚
  甲以白糖为砒霜投入他人水杯 → 不能犯未遂，可罚
```

### 二、反向禁止错误/反向事实错误

> 注：本节所称"不能犯未遂"，中国《刑法》第23条对未遂仅作一般规定，未对不能犯设独立条款。学理多数说认为绝对不能犯不可罚，相对不能犯仍按未遂处罚。

```
反向禁止错误 = 假想犯罪：以为违法实际合法 → 不可罚
反向事实错误 = 不能犯未遂：以为能实现实际不能 → 学理多数说：相对不能犯按未遂处罚；绝对不能犯不可罚
```

### 三、假想防卫过当

#### 概念

行为人误以为存在正当防卫情状，并在此想象基础上实施了过当的防卫。

#### 处理难度

```
复杂性分析：
- 客观上：无不法侵害 → 无正当防卫情状
- 主观上：行为人相信有侵害 → 容许构成要件错误
- 进一步：即便在行为人想象中，其防卫行为也超出了必要限度

问题：这种"双重错误+过当"如何处理？
```

#### 处理方案

```
方案一：分层处理

  第一层：必要限度内的部分
    → 按"容许构成要件错误"处理
    → 排除故意犯处罚

  第二层：超出必要限度的部分（过当部分）
    → 不能援引"防卫过当减免"（因客观上无防卫情状）
    → 也不能按容许构成要件错误处理（因即便事实前提存在也超出许可范围）
    → 对过当部分成立故意犯

方案二：整体评价

  → 行为人主观上处于"恐惧/慌乱/惊吓"状态
  → 可能基于期待可能性降低而减轻/免除责任
  → 具体需个案判断

本程式库立场：
  原则采分层处理，但注意期待可能性的补充适用。
```

## 错误理论总结对照表

| 错误类型 | 体系位置 | 法律效果 | 实例 |
|----------|----------|----------|------|
| 构成要件事实错误 | 主观构成要件 | 排除故意→可能过失 | 不知对方是人 |
| 对象错误 | 主观构成要件 | 不排除故意 | 误A为B而杀之 |
| 打击错误 | 归责对应（争议） | 具体符合说：未遂+过失竞合 | 射A中B |
| 因果流程偏离 | 归责对应 | 可预见范围内不排除既遂 | 意外死因 |
| 容许构成要件错误 | 有责性 | 排除故意犯处罚 | 假想防卫 |
| 禁止错误（不可避免） | 有责性 | 阻却责任 | 不知行为违法 |
| 禁止错误（可避免） | 有责性 | 不阻却，可减轻 | 应知而未查 |
| 假想犯罪 | — | 不可罚 | 以为违法实际合法 |
