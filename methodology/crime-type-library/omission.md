# 不作为犯

> **体系定位**：不作为犯是作为犯的镜像。不作为犯违反的不是"禁止规范"，而是"命令规范"——法律命令行为人积极行为以保护法益或监督危险源。

---

## 总论：作为与不作为的区分

### 一、区分标准

| 判断维度 | 作为 | 不作为 |
|----------|------|--------|
| 行为本质 | 以能量投入积极创设因果流程 | 未为法律所期待的积极行为 |
| 规范类型 | 违反禁止规范 | 违反命令规范 |
| 因果关系 | 真实因果关系（条件说） | 假设因果关系（若作为则结果可避免） |
| 典型形态 | "做了不应做的事" | "未做应做的事" |

### 二、区分困难情形

#### 情形 1：积极介入后的不阻止

- 行为人先制造了危险（作为），后未阻止结果发生（不作为）
- **处理**：若前行为已构成独立的作为犯，以作为犯处理；若前行为本身不直接致果，则检验不作为犯（先行行为产生保证人义务）

#### 情形 2：中止救助行为

- 行为人开始救助后又中途停止
- **处理**：
  - 若尚未创设新的依赖关系 → 不改变原有法律地位
  - 若已使被害人放弃自救或其他救助可能 → 可能构成不作为犯

#### 情形 3：作为与不作为竞合

- 同一事实可从两个角度评价（如关闭他人的生命维持装置）
- **处理原则**：作为优先（评价重点说）
  - 若行为重点在能量投入 → 以作为犯处理
  - 若行为重点在期待义务的不履行 → 以不作为犯处理

#### 情形 4：持续状态中的不中断

- 例：非法拘禁的持续
- **处理**：最初的积极行为为"作为"，维持状态可构成"不作为"（不解除拘禁）

---

## 第一部分：真正不作为犯

### 概念

- **定义**：刑法条文明确规定以"不作为"为构成要件行为的犯罪
- **特征**：条文本身即描述了不履行特定义务的行为
- **不需要保证人地位**：义务直接来源于法律条文

### 检视程式

```
I. 构成要件符合性
   A. 行为主体
   B. 特定情状存在（法定的作为义务触发条件）
   C. 未履行法定义务（不作为）
   D. 作为可能性（行为人有能力作为）
   E. 主观构成要件（故意或过失，视条文要求）

II. 违法性
III. 有责性
IV. 其他要件
```

### 典型罪名

| 罪名 | 义务内容 | 法律依据 |
|------|----------|----------|
| 遗弃罪 | 对年老/年幼/患病等无自救能力者的扶养义务 | 《刑法》第261条 |
| 拒不救助伤病军人罪 | 战时对伤病军人的救助义务 | 《刑法》第445条 |
| 不报告传染病疫情罪 | 从事传染病防治者的报告义务 | 相关行政法规 |

## 第二部分：不真正不作为犯

### 概念

- **定义**：以不作为的方式实现原本由作为犯条文规定的构成要件
- **核心问题**：在何种条件下，"不作为"可以与"作为"等价
- **关键要素**：保证人地位——赋予行为人积极防止结果发生的特别义务

### 完整检视程式

```
前置检验：
  1. 不作为性确认（行为为不作为而非作为）
  2. 补充性条款（法律对此类不作为有处罚可能性）

I.   构成要件符合性
     A. 客观构成要件
         1. 构成要件结果发生
         2. 可能防止结果的行为确定
         3. 作为可能性
         4. 假设因果关系
         5. 保证人地位
         6. 等置性
     B. 主观构成要件
         1. 故意（覆盖全部客观要素）
         2. 主观超过要素（如有）

II.  违法性
     A. 标准正当化事由
     B. 义务冲突

III. 有责性
     A. 责任能力
     B. 违法性认识可能性/诫命错误
     C. 期待可能性
     D. 责任故意

IV.  其他要件
```

---

### I. 构成要件符合性

#### A. 客观构成要件

##### 1. 构成要件结果发生

- 不真正不作为犯原则上为结果犯
- 须确认法定结果已发生（死亡、伤害、财产损失等）
- 无结果发生时：可能成立不作为的未遂（如有犯罪决意+不作为义务违反+着手）

##### 2. 可能防止结果的行为确定

- 确定行为人应为何种具体行为以防止结果
- 即：法律所期待的具体作为内容是什么？
- 须具体化到可操作的行为层面

##### 3. 作为可能性

```
检验要点：
- 物理可能性：行为人在当时情境下是否有物理能力采取行动？
- 认知可能性：行为人是否知悉或可以知悉需要行动的情况？
- 排除情形：
  - 物理不能：如被锁在房间内无法施救
  - 认知不能：如完全不知危险情况的存在
  - 注意：能力不足≠不可能（可借助他人/工具的情况下仍有可能）
```

##### 4. 假设因果关系

> **公式**："若行为人采取了法律期待的行为，结果是否能以接近确定的概率避免？"

| 判断标准 | 内容 |
|----------|------|
| 判断公式 | 补充考量法（在思维上补充加入期待行为） |
| 确定性要求 | "接近确定"可避免 |
| 不确定时 | 适用"存疑有利于被告"原则 → 否定因果关系 |
| 风险降低争议 | 与过失犯中合义务替代行为理论相似 |

**注意**：不作为犯的因果关系本质上是"假设的"——我们无法观察到"若作为"的真实结果，只能做规范性判断。

##### 5. 保证人地位

> **核心问题**：本要素是不真正不作为犯区别于真正不作为犯和故意作为犯的关键

###### 保证人地位的功能

- 限制不作为犯的处罚范围——非任何人的不作为均可与作为等价
- 仅对法益负有特别保护义务之人的不作为，方可构成不真正不作为犯
- 保证人地位将一般的道德义务提升为刑法上的作为义务

###### 保证人类型体系（功能二分说）

```
保证人地位
├── 保护义务型保证人
│   功能：保护特定法益免受一切危险
│   方向：从法益出发，向外防御各种危险
│
└── 监督义务型保证人
    功能：控制特定危险源，防止其侵害他人法益
    方向：从危险源出发，向外防止其侵害
```

###### A. 保护义务型保证人

| 类型 | 内涵 | 典型情形 |
|------|------|----------|
| **紧密人身关系** | 基于家庭/亲密关系产生的保护义务 | 父母对子女、配偶之间、同居伴侣之间 |
| **自愿承担** | 自愿接管对他人法益的保护 | 保姆对幼儿、登山向导对游客、医生对患者 |
| **先行行为/危险前行为** | 因先前行为创设危险而负有防止结果义务 | 见下文专题详论 |
| **危险共同体** | 共同面对危险的团体成员之间的互救义务 | 登山团队、探险小组 |
| **紧密生活共同体** | 实际共同生活产生的保护义务 | 同居者之间 |

###### B. 监督义务型保证人

| 类型 | 内涵 | 典型情形 |
|------|------|----------|
| **危险源监管** | 对自己支配领域内的危险源负有控制义务 | 动物饲主、危险设施所有人、有缺陷产品生产者 |
| **第三人监管** | 对受自己监管的人员的行为负责 | 父母对未成年子女、雇主对员工在职务范围内的行为 |
| **对自己先前创设的危险源的控制** | 自己制造了危险状态后的控制义务 | 挖坑未设警示标志者 |

###### 保证人地位的限制

- **保护范围限制**：保证人义务仅限于特定法益/特定危险范围
- **合理性限制**：不要求保证人牺牲自己的重大利益
- **可行性限制**：须在保证人的实际能力范围内

##### 6. 等置性

> 不作为是否在规范评价上等同于作为实施？

```
等置性检验：
1. 不真正不作为犯以"不作为"实现作为犯条文的构成要件
2. 须检验：该不作为在社会意义上是否等同于"积极作为"
3. 特别重要的场合：
   a. 具有特殊行为方式要求的犯罪（如诈骗罪的"欺骗行为"）
      → 不告知真相的不作为是否等同于积极陈述虚假事实？
   b. 亲手犯
      → 一般认为不作为不能构成亲手犯
4. 判断标准：社会评价上的可比性、非难程度的等同性
```

---

#### B. 主观构成要件

##### 1. 故意

- 故意须覆盖所有客观构成要件要素：
  - 认识到结果发生的危险
  - 认识到自己有能力行为
  - 认识到自己的保证人地位（至少"外行人平行评价"层面的认知）
  - 认识到不作为将导致结果发生
- 意志要素：容忍/接受结果发生（间接故意即可）

##### 2. 对保证人地位事实的认知

- 不要求行为人精确知晓法律概念
- 要求：认识到赋予其保证人地位的事实基础
- 例：母亲知道那是自己的孩子且孩子面临危险 → 对保证人地位事实有认知
- 不知道自己有保证人义务 → 可能构成禁止错误/诫命错误（学理处理；中国《刑法》对违法性认识无明文，本节按学理通说在有责性层处理）

### II. 违法性

#### A. 标准正当化事由

与作为犯相同：正当防卫、紧急避险、被害人承诺等均可适用。

**不作为犯中正当化事由的特殊问题：**

| 问题 | 处理 |
|------|------|
| 正当防卫 | 可以通过不作为实施正当防卫（如保证人拒绝救助侵害者） |
| 紧急避险 | 保证人为保护更大利益而牺牲较小利益 |
| 被害人承诺 | 被害人放弃要求保证人保护的请求 |

#### B. 义务冲突

> **不作为犯独有的正当化事由**

```
适用情境：
  行为人同时负有数个作为义务，但无法同时履行全部义务

处理规则：
  1. 等价义务冲突（如同时有两个溺水者需要救助）：
     → 履行任何一项义务即为合法
     → 对未履行的义务不承担不作为犯的责任
  2. 不等价义务冲突（一项义务高于另一项）：
     → 须履行较高义务
     → 对较低义务的不履行被正当化
  3. 作为义务与不作为义务的冲突：
     → 原则上不作为义务优先（不得为了救助而主动侵害他人法益）
     → 例外：紧急避险可以突破

判断标准：
  - 法益位阶（生命 > 身体 > 财产）
  - 危险紧迫程度
  - 成功救助的概率
```

---

### III. 有责性

#### A. 责任能力

与作为犯相同。

#### B. 违法性认识可能性/诫命错误

**诫命错误的类型：**

| 错误类型 | 内容 | 处理 |
|----------|------|------|
| 不知有作为义务 | 行为人不知自己负有保证人义务 | 按学理上禁止错误规则处理 |
| 误认义务范围 | 行为人误认义务范围窄于实际 | 同上 |
| 误认作为内容 | 行为人误解应为何种行为 | 同上 |

**处理方案：**
- 不可避免的诫命错误 → 阻却责任
- 可避免的诫命错误 → 不阻却但可减轻
- 判断标准：行为人是否有义务且有可能了解到相关法律义务的存在

#### C. 期待可能性

- 不作为犯中期待可能性具有特别重要的功能
- 问题：能否期待保证人在具体情境下冒险履行义务？
- 例：大火中是否能期待父亲冲入火场救子？
- 通说：不要求自我牺牲，但可期待承担一定程度的风险

#### D. 责任故意/容许构成要件错误

与作为犯相同——若行为人误以为存在免除其作为义务的正当化情状（如误以为被害人已同意放弃保护），按容许构成要件错误处理。

## 专题：先行行为与保证人地位

### 一、基本原理

> 因先前行为制造或增大了法益面临的危险的人，负有防止该危险实现的保证人义务。

### 二、先行行为产生保证人义务的要件

```
1. 先行行为的存在
2. 先行行为制造了法益面临的紧迫危险
3. 该危险与最终结果之间具有关联性
4. 行为人认识到危险的存在（或可认识到）
```

### 三、争议问题：先行行为须否违法？

| 学说 | 主张 | 理由 |
|------|------|------|
| 广义说 | 任何制造危险的行为均可产生保证人义务 | 危险制造即产生控制义务 |
| 义务违反说 | 先行行为须违反注意义务 | 合法行为不应额外负担义务 |
| 违法性说 | 先行行为须为违法行为 | 与紧急避险的评价一致 |
| 限缩说（通说） | 先行行为须为"义务违反且制造了不被容许的风险"的行为 | 兼顾保护需要与自由保障 |

### 四、典型争议案例

| 案例 | 争议焦点 | 通说结论 |
|------|----------|----------|
| 合法正当防卫后侵害者生命垂危 | 正当防卫行为是否产生救助义务？ | 否（合法行为不产生保证人义务） |
| 合法驾驶中发生不可避免的事故 | 合法驾驶行为是否产生救助义务？ | 争议（多数肯定，基于支配关系说） |
| 犯罪行为导致被害人陷入危险 | 犯罪行为是否产生保证人义务？ | 是（违法行为当然产生保证人义务） |
| 在自己的土地上挖坑未设标志 | 合法利用自己的财产 | 是（对危险源的管理义务） |

### 五、先行行为义务的范围限制

- 保证人义务的范围受先行行为制造的具体危险类型限制
- 先行行为创设的特定风险实现 → 义务范围内
- 与先行行为无关的独立新危险 → 义务范围外
- 例：因过失伤害他人后，对伤害结果负有救助义务，但对被害人在医院遭遇的独立侵害不负保证人义务

## 专题：继续犯中的保证人义务

### 一、继续犯的概念

- 继续犯：犯罪行为在一定时间内持续存在
- 例：非法拘禁、非法持有枪支、窝藏犯

### 二、与不作为犯的关系

```
继续犯的结构：
  初始行为（作为）→ 持续状态（可评价为不作为）→ 终了行为

分析：
  1. 初始的积极行为构成"作为犯"
  2. 维持违法状态的"不中断"可构成"不作为"
  3. 此时行为人基于先行行为（最初的作为）负有保证人义务
  4. 不解除违法状态 = 不作为犯
```

### 三、继续犯的括弧效应

- 继续犯的持续期间内实施的其他犯罪，与继续犯构成想象竞合
- 理由：继续犯的整个持续过程视为"一个行为"
- 详见 `concurrence.md`

## 专题：不作为犯的未遂

### 一、未遂的可能性

不真正不作为犯可以成立未遂：
- 行为人有保证人义务
- 有犯罪决意（故意不救助以使结果发生）
- 已"着手"（不作为的着手）
- 结果未发生（由于偶然因素或第三人介入）

### 二、不作为犯着手的判断

```
着手时点判断：
  - 非以"开始不作为"为着手（否则义务产生时即为着手，过于提前）
  - 通说：以法益陷入紧迫的具体危险为着手时点
  - 即：行为人的不作为已使法益面临直接、紧迫的危险

例：
  - 母亲决定不喂食婴儿使其死亡
  - 着手时点 ≠ 第一次没喂食
  - 着手时点 = 婴儿已因未进食而陷入生命危险的时点
```

### 三、不作为犯的中止

- 中止行为：在结果发生前积极履行保证人义务
- 自动性判断与作为犯相同

## 专题：过失不作为犯

### 检视程式的修正

```
过失不作为犯 = 不真正不作为犯 + 过失犯

I. 构成要件符合性
   A. 客观构成要件：
      1-6. 同故意不作为犯（结果+可能行为+作为可能性+假设因果+保证人+等置性）
      7. 客观注意义务违反：保证人未尽到应有注意
   B. [无主观构成要件]

II. 违法性
III. 有责性
   A-C. 同一般有责性要件
   D. 主观注意义务违反（个人标准）

特殊问题：
  - 保证人对危险的无认识 → 是否违反了"结果预见义务"？
  - 对保证人地位事实的无认识 → 过失的"事实错误"？
```

```
     └─────────────────┬─────────────────────────┘
                       │ 无正当化
                       ▼
     ┌───────────────────────────────────────────┐
     │  III. 有责性                               │
     │  责任能力 + 诫命错误 + 期待可能性          │
     └─────────────────┬─────────────────────────┘
                       │ (+)
                       ▼
     ┌───────────────────────────────────────────┐
     │  结论：不作为犯成立                        │
     └───────────────────────────────────────────┘
```
