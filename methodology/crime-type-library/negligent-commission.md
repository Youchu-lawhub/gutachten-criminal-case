# 过失作为犯

> **体系定位**：本检视程式以**二阶的过失犯论**为基础，将客观注意义务违反定位于构成要件符合性层，将主观注意义务违反定位于有责性层。

> **关于过失共同犯罪**：中国《刑法》第25条第2款明确规定"二人以上共同过失犯罪，不以共同犯罪论处"。因此，学理上的"过失共同正犯"在中国法上原则上不成立，应当根据各自所犯的罪分别处罚。本程式库讨论过失犯时以单独犯为基础。

---

## 完整检视程式

```
前置确认：
  - 该罪名的过失犯处罚规定存在（"法律有规定的才负刑事责任"——《刑法》第15条第2款）

I.   构成要件符合性
     A. 客观构成要件
         1. 行为主体
         2. 构成要件行为
         3. 行为对象
         4. 构成要件结果
         5. 因果关系与客观归责
         6. 客观注意义务违反
     B. [无主观构成要件——过失犯不要求"故意"]

II.  违法性

III. 有责性
     A. 责任能力
     B. 违法性认识可能性
     C. 期待可能性
     D. 主观注意义务违反/责任过失

IV.  其他要件
```

---

## I. 构成要件符合性

### A. 客观构成要件

#### 1-4. 行为主体、行为、对象、结果

与故意作为犯相同——参见 `intentional-commission.md` I.A.1-4。

**特别注意：**
- 过失犯原则上为结果犯，须有法定结果发生
- 无结果发生时不成立过失犯（无"过失未遂"概念）
- 行为犯中过失犯极为罕见

#### 5. 因果关系与客观归责

##### 事实归因（条件说）

与故意犯相同——"若无行为人的注意义务违反行为，结果是否不会发生？"

##### 客观归责的特殊问题

过失犯中客观归责论的适用具有独特的重要性：

| 问题 | 在过失犯中的体现 |
|------|-----------------|
| 制造不被容许的风险 | = 客观注意义务违反（两者高度重叠） |
| 风险实现 | 违反义务的行为创设的风险在具体结果中实现 |
| 规范保护目的 | 所违反的注意义务是否旨在防止此类结果？ |
| 义务违反关联 | 合义务替代行为能否避免结果？ |

#### 6. 客观注意义务违反

> **核心要素**：过失犯的不法核心在于违反客观注意义务

##### a. 注意义务的确定

**注意义务的来源：**

| 来源类型 | 说明 | 举例 |
|----------|------|------|
| 法律法规 | 明确规定的行为规则 | 道路交通法规、安全生产法规 |
| 行政规章/技术标准 | 行业准则、操作规程 | 医疗指南、建筑规范 |
| 习惯/惯例 | 特定领域通行做法 | 行业惯例、交易习惯 |
| 一般注意义务 | 无具体规范时的合理谨慎 | 社会生活中的一般注意 |

**注意义务的内涵（双重义务）：**

```
客观注意义务
├── 结果预见义务
│   └── 预见结果发生的可能性
└── 结果避免义务
    └── 采取适当措施防止结果发生
```

##### b. 结果预见义务

- **判断标准**：以处于行为人相同情境的**理性一般人**为准
- **认识基础**：行为时已知或可知的全部情况
- **预见程度**：不要求精确预见因果流程，对结果发生的一般方向有预见即可
- **抽象危险认知**：认识到行为类型上的危险性即可，不需认识到具体危险

**客观可预见性：**

| 判断要素 | 内容 |
|----------|------|
| 判断主体 | 理性一般人（非行为人本人） |
| 判断时点 | 行为时（ex ante） |
| 判断基础 | 行为时可利用的全部信息 |
| 预见对象 | 结果发生的"大致轮廓" |
| 否定情形 | 结果发生属于极端异常、超出一般生活经验 |

##### c. 结果避免义务

- 行为人应采取何种措施以避免结果发生
- 衡量因素：
  - 预见到的危险程度
  - 法益的重要性
  - 采取避免措施的可能性与成本
  - 行为的社会效用

##### d. 义务违反关联/风险实现

> **核心问题**：行为人违反注意义务所创设的特定风险，是否正是在具体结果中得到了实现？

**检验步骤：**

```
1. 确认行为人违反了注意义务
2. 确认具体结果发生
3. 追问：若行为人遵守注意义务（合义务替代行为），结果是否可以避免？
   - 确定可以避免 → 义务违反关联肯定
   - 确定无法避免 → 义务违反关联否定 → 不成立过失犯
   - 不确定（风险升高） → 理论争议（见下文）
```

**合义务替代行为理论争议：**

| 学说 | 主张 | 结论 |
|------|------|------|
| 回避可能性说（通说） | 合义务行为须"确定"能避免结果 | 不确定时否定归责 |
| 风险升高说 | 义务违反行为"升高了"结果发生的风险即可 | 不确定时仍肯定归责 |

**本程式库立场**：原则采**回避可能性说**，但标注风险升高说观点。

---

### B. 主观构成要件

过失犯在构成要件层**不存在主观构成要件**：
- 过失不是故意的"弱化形态"，而是一种完全不同的归责类型
- 主观面（行为人个人能力）在有责性层处理
- 这是二阶的过失犯论与传统一阶论的根本区别

---

## II. 违法性

### 检验方式

与故意犯相同——消极检验有无正当化事由。

### 过失犯中正当化事由的特殊问题

| 问题 | 处理 |
|------|------|
| 正当防卫 | 过失侵害行为可以构成"不法侵害"——他人可对之正当防卫 |
| 过失行为人的正当防卫 | 理论上可能，但实际上难以想象（过失行为人通常不知自己在侵害法益） |
| 紧急避险 | 过失犯中紧急避险的适用与故意犯无异 |
| 容许风险 | 严格而言已在构成要件层排除（不构成注意义务违反） |

### 主观正当化要素的问题

- 二元不法论要求正当化事由具备主观要素
- 过失犯中：行为人通常不知自己的行为侵害法益，遑论主观防卫意识
- **解决方案**：过失犯中对主观正当化要素的要求应适当放宽——只要客观正当化情状存在，即便行为人不知，也可考虑排除违法性（理论争议）

---

## III. 有责性

### A. 责任能力

与故意犯相同——参见 `intentional-commission.md` III.A。

### B. 违法性认识可能性

- 过失犯中同样要求行为人具有违法性认识的可能性
- 禁止错误在过失犯中的适用（学理处理）：行为人不知自己负有注意义务 → 可能构成禁止错误
- 但通常：注意义务的存在本身就要求行为人去了解相关规范 → 大多数情况下禁止错误"可避免"

### C. 期待可能性

- 是否能期待行为人在具体情境下遵守注意义务
- 极端情形下可阻却责任
- 例：长时间连续工作后的疲劳驾驶导致事故

### D. 主观注意义务违反/责任过失

> **核心功能**：在有责性层检验行为人**个人是否有能力**遵守客观注意义务

##### 判断标准：行为人个人标准（非一般人标准）

```
检验内容：
1. 主观预见可能性：
   - 以行为人个人的认知能力、知识水平、经验为标准
   - 行为人个人能否预见结果发生？

2. 主观避免可能性：
   - 以行为人个人的身体能力、精神状态为标准
   - 行为人个人能否采取避免措施？

3. 个人能力充分性：
   - 行为人是否具备遵守注意义务所需的个人能力？
   - 新手/专家的不同标准
```

**客观注意义务违反 vs 主观注意义务违反的关系：**

| 层次 | 标准 | 功能 | 位置 |
|------|------|------|------|
| 客观注意义务违反 | 理性一般人（客观化） | 界定不法 | 构成要件符合性 |
| 主观注意义务违反 | 行为人个人能力 | 界定责任 | 有责性 |

**例外情形：**
- 行为人能力**高于**一般人：以行为人个人标准（否则允许有能力者不尽全力）
- 行为人能力**低于**一般人：不成立主观注意义务违反 → 阻却责任

---

## IV. 其他要件

与故意犯相同——客观处罚条件、刑罚解除事由（如有）。

## 专题：间接故意与有认识过失的区分

### 理论框架

| 理论 | 区分标准 | 评价 |
|------|----------|------|
| 容任说【本程式库采】 | 行为人是否内心接受/容忍结果发生 | 通说，兼顾认识与意志 |
| 认识说/盖然性说 | 行为人是否认识到结果发生的高度盖然性 | 偏重认知，但难以划定"高度"标准 |
| 无所谓说 | 行为人对结果是否漠不关心 | 概念模糊 |
| 认真对待说 | 行为人是否认真对待结果发生的可能性 | 实质上与容任说接近 |

### 容任说的具体适用

```
间接故意（dolus eventualis）：
  认识：认识到结果发生的可能性
  意志：认真对待该可能性 + 内心接受/容忍/放任
  态度："即便发生也在所不惜"

有认识过失：
  认识：认识到结果发生的可能性
  意志：真诚信赖结果不会发生 / 轻信能够避免
  态度："不会发生的"/"我能控制住"
```

### 实务判断的辅助指标

| 指标 | 倾向间接故意 | 倾向有认识过失 |
|------|-------------|---------------|
| 行为危险程度 | 极高危险行为（如高速逆行） | 轻度风险行为 |
| 避免措施 | 未采取任何避免措施 | 采取了一定预防手段 |
| 对自身的危险 | 行为对自身无危险 | 行为对自身同样危险 |
| 行为后态度 | 无救助行为 | 积极救助 |
| 动机关联 | 结果发生不妨碍其目的实现 | 结果发生违反其利益 |

## 专题：管制领域与非管制领域的注意义务确定

### 管制领域

- **特征**：存在明确的法律法规、行政规章、技术标准
- **注意义务来源**：直接援引成文规范
- **违反判断**：违反具体规定即可推定违反注意义务
- **例外**：遵守规定但因特殊情况仍应采取额外注意的情形

| 领域 | 典型规范 | 举例 |
|------|----------|------|
| 道路交通 | 道路交通安全法及实施条例 | 限速规定、信号灯规则 |
| 医疗 | 诊疗规范、临床指南 | 术前检查义务、用药规范 |
| 建筑施工 | 安全生产法、建设标准 | 防护设施要求 |
| 食品安全 | 食品安全法及标准 | 检验检测义务 |

### 非管制领域

- **特征**：无具体成文规范
- **注意义务确定方法**：
  1. 参照行业惯例/一般做法
  2. 利益衡量（法益重要性 vs 行为效用 vs 预防成本）
  3. "理性谨慎人"在该情境下会如何行为
- **注意**：法官具有较大的裁量空间

## 专题：结果避免可能性与合义务替代行为

### 问题本质

当确认行为人违反了注意义务，但即使遵守义务结果仍然可能发生时：

```
行为人违反限速（时速80km/h区域开110km/h）→ 撞死行人
追问：即便时速80km/h行驶，该行人是否仍会被撞死？
```

### 判断方法

```
Step 1: 构建"合义务替代行为"
        → 假设行为人完全遵守注意义务
Step 2: 判断在该假设下结果是否仍然发生
        → 确定避免 / 确定仍发生 / 不确定
Step 3: 适用结论
        → 确定避免：肯定义务违反关联
        → 确定仍发生：否定义务违反关联 → 不成立
        → 不确定：理论争议（见上文学说对立）
```

## 专题：规范保护目的限制

### 适用规则

- 所违反的注意义务/规范的保护目的必须涵盖实际发生的结果
- 若结果类型不在该规范的保护范围内，否定客观归责

### 典型案例

| 情形 | 规范目的 | 结论 |
|------|----------|------|
| 超速行驶 → 路人被溅水弄脏 | 限速旨在防止碰撞伤亡 | 弄脏不在保护范围内 |
| 药店无处方卖药 → 买药者按说明服用仍过敏死亡 | 处方制度旨在防止用药错误 | 正确用药的过敏不在保护范围 |
| 违规横穿马路 → 被合规行驶的车辆撞伤 | 禁止横穿旨在保护横穿者自身 | 在保护范围内 |

## 专题：被害人自我答责

### 过失犯中的特殊意义

在过失犯中，被害人自身的行为对结果的贡献具有特别重要的归责排除功能：

```
排除归责的条件：
1. 被害人系完全自我答责地参与危险
2. 被害人对危险有充分认识
3. 被害人的决定系自由做出（非受欺骗/强制）
4. 行为人对被害人不负有特别的保护义务

排除效果：
→ 结果不能归责于行为人
→ 过失犯不成立
```

### 典型情形

| 情形 | 被害人自我答责？ | 理由 |
|------|-----------------|------|
| 自愿搭乘醉酒司机的车 | 争议（多数肯定） | 被害人明知风险仍自愿参加 |
| 贩毒者出售毒品，买家自行注射过量 | 争议 | 涉及毒品犯罪的特殊评价 |
| 被害人明知传染病风险仍自愿与他人性行为 | 肯定 | 被害人自主承担风险 |
| 未成年被害人参加危险游戏 | 否定 | 缺乏完全的自我答责能力 |

## 专题：信赖原则

### 概念

行为人在参与社会交往时，原则上可以信赖他人同样遵守规则，无需随时为他人的违规行为做准备。

### 适用条件

```
信赖原则适用 = 排除/限缩注意义务

适用条件：
1. 存在分工协作的社会交往领域
2. 行为人自身遵守了相关规则
3. 无具体情况表明他人将违规

不适用情形：
1. 已有迹象表明他人将违规
2. 对方为儿童、老人、醉酒者等明显无法信赖之人
3. 行为人自身已违规
4. 行为人对对方负有监督义务
```

### 典型领域

| 领域 | 信赖内容 | 限制 |
|------|----------|------|
| 道路交通 | 信赖他人遵守交规 | 有明显违规迹象时失效 |
| 医疗分工 | 医生信赖护士正确执行医嘱 | 有明显异常时须自行核查 |
| 建筑施工 | 信赖各工种各尽其责 | 明知对方能力不足时失效 |

## 专题：过失结果加重犯

### 与纯粹过失犯的关系

- 纯粹过失犯：整体以过失实施（如过失致人死亡）
- 结果加重犯中的过失：基本犯可能是故意，但对加重结果仅有过失

### 交叉关系

```
过失作为犯 ←——→ 结果加重犯
                    ↑
                    │ 对加重结果的过失部分
                    │ 适用本检视程式的注意义务分析
                    │ 详见 result-aggravated.md
```
