# 共同犯罪

> **体系定位**：本程式库以**区分制**严格区分正犯与共犯，所讨论的"共同正犯/间接正犯/教唆犯/帮助犯"五分法属学理工具；最终归责须落回中国《刑法》第 25–29 条法定的"主犯/从犯/胁从犯/教唆犯"四分体系。第 25 条第 2 款明文排除过失共同犯罪。

---

## A. 基本原则

### 一、分类方法论

**先分工分类后作用分类：**

```
第一步：按"分工"确定参与形态
  ┌─────────────────────────────────┐
  │ 正犯                            │
  │  ├── 单独正犯                   │
  │  ├── 共同正犯                   │
  │  └── 间接正犯                   │
  ├─────────────────────────────────┤
  │ 共犯                            │
  │  ├── 教唆犯                     │
  │  └── 帮助犯                     │
  └─────────────────────────────────┘

第二步：按"作用"确定刑罚轻重
  → 主犯/从犯的认定
  → 量刑层面的考量
```

### 二、检验顺序原则

**先正犯后共犯：**
- 理由：共犯从属于正犯的不法行为
- 操作：先检验正犯是否成立犯罪（至少构成要件符合性 + 违法性），再检验共犯

### 三、正犯与共犯的区分标准

#### 犯罪事实支配理论

| 支配类型 | 适用参与形态 | 内涵 |
|----------|-------------|------|
| 行为支配 | 单独正犯 | 亲自实施构成要件行为 |
| 功能性支配 | 共同正犯 | 基于分工实施中的不可或缺地位 |
| 意思支配 | 间接正犯 | 通过操控他人实现犯罪 |
| 无支配 | 共犯 | 仅促进/推动他人犯罪 |

#### 重要作用说（中国通说补充）

- 在正犯与共犯的区分上，学理上结合犯罪事实支配理论与"重要作用说"
- 对犯罪实现起"重要/不可或缺作用"者为正犯
- 仅起"辅助/次要作用"者为共犯（对应中国《刑法》第27条的从犯之"辅助作用"类型）

### 四、限制从属性

```
共犯的成立条件：
  正犯行为须满足：
  ✓ 构成要件符合性
  ✓ 违法性
  ✗ 不要求有责性—— 即正犯不必有责

含义：
  - 教唆/帮助无责任能力者犯罪 → 可能构成间接正犯（而非教唆/帮助犯）
  - 教唆/帮助有构成要件符合性+违法性但无责任的人 → 教唆/帮助犯可成立
  - 正犯行为不构成犯罪（无构成要件符合性或有正当化事由）→ 共犯不成立
```

## B. 共同正犯

### 检视程式

```
I. 构成要件符合性
   A. 客观构成要件
      1. 构成要件结果发生（如结果犯）
      2. 各共同正犯的身份要件满足（如身份犯）
      3. 共同正犯要件：
         a. 共同犯罪计划
         b. 各人参与实施
         c. 功能性犯罪事实支配
      4. 相互归责：
         → 全部参与者的行为和结果归责于每一人
   B. 主观构成要件
      1. 故意（覆盖全部客观构成要件 + 共同实施的意思）
      2. 主观超过要素（如有）

II. 违法性
    → 对每一共同正犯个别检验

III. 有责性
    → 对每一共同正犯个别检验
```

### 共同正犯要件详解

#### a. 共同犯罪计划

```
要素：
1. 意思联络：
   - 各参与者知悉彼此的参与
   - 达成共同实施犯罪的合意
   - 形式不限：明示/默示/暗示均可
   - 时点：至迟在实行阶段前形成

2. 计划内容：
   - 对犯罪目标的共同认识
   - 对各人分工角色的大致了解
   - 不要求对具体细节完全一致

3. 特殊情形：
   - 现场临时形成的共同计划（承继的共同正犯）
   - 相继加入的共同犯罪计划
```

#### b. 各人参与实施

```
要求：
- 各参与者须对犯罪的实施提供贡献
- 贡献须在实行阶段作出（通说）
  - 争议：预备阶段的贡献是否足够？
  - 多数说：预备阶段贡献 + 共同计划 = 仍可构成共同正犯

贡献类型：
- 直接实施构成要件行为的一部分
- 虽未亲自实施构成要件行为，但提供了不可或缺的功能性贡献
  （如望风、驾车、制压被害人）
```

#### c. 功能性犯罪事实支配

```
核心标准：
- "部分实行全部责任"
- 各参与者的贡献相互利用、相互补充
- 每一参与者的贡献对犯罪整体成功而言具有"不可或缺性"
- 移除任一参与者的贡献，犯罪计划即无法实现

判断因素：
┌─────────────────────────────────────────┐
│ (+) 有利于认定共同正犯的因素：           │
│   - 对犯罪计划的参与制定                  │
│   - 在实行阶段的核心地位                  │
│   - 对犯罪是否进行的决定权               │
│   - 分得犯罪收益                         │
│                                          │
│ (-) 不利于认定共同正犯的因素：           │
│   - 仅提供辅助性帮助                     │
│   - 可被他人轻易替代                     │
│   - 对犯罪进程无决定性影响               │
└─────────────────────────────────────────┘
```

### 共同正犯的特殊问题

#### 承继的共同正犯

| 问题 | 处理 |
|------|------|
| 后加入者对先前行为是否负责？ | 仅对加入后的行为与结果负责（通说） |
| 先前行为已造成的结果 | 不可归责于后加入者 |
| 后加入者明知前行为事实 | 不等于对前行为的"共同实施" |

#### 片面共同正犯

- **概念**：一方有共同犯罪的意思，另一方不知情
- **通说**：否认片面共同正犯——共同犯罪计划要求双方合意
- **少数说**：肯定片面共同正犯在特定条件下可成立
- **本程式库立场**：原则否认，但标注争议

#### 过失共同正犯

《刑法》第 25 条第 2 款"二人以上共同过失犯罪，不以共同犯罪论处"明文排除，应分别处罚。

- **通说**：否定——过失犯无"共同犯罪决意"，各人仅就自身注意义务违反负责
- **少数有力说**：肯定——对共同危险行为中无法确定具体致果者的处理需要
- **本程式库立场**：原则否认，注明争议存在

## C. 间接正犯

### 检视程式

```
I. 构成要件符合性
   A. 客观构成要件
      1. 构成要件结果发生
      2. 间接正犯要件：
         a. 前台人（工具）的行为存在
         b. 前台人行为欠缺某犯罪成立要素
         c. 后台人对犯罪事实具有支配性
      3. 因果关系：后台人行为 → 前台人行为 → 结果
   B. 主观构成要件
      1. 故意（认识并利用前台人作为工具）
      2. 支配意思

II. 违法性
III. 有责性
```

### 意思支配的类型

| 支配类型 | 内涵 | 典型情形 |
|----------|------|----------|
| **强制支配** | 以强制使前台人丧失意志自由 | 以死亡威胁迫使他人实施犯罪 |
| **错误支配** | 利用前台人的认知错误 | 使他人误以为在做合法行为 |
| **组织支配** | 利用组织权力结构中的可替代性 | 犯罪组织领导者下达指令 |
| **能力欠缺利用** | 利用前台人欠缺责任能力 | 利用精神病人/幼儿实施犯罪 |

### 各支配类型详解

#### 强制支配

```
条件：
1. 后台人对前台人施加强制（vis compulsiva）
2. 强制程度达到使前台人的意志决定不再自由
3. 但前台人仍有物理上的行为可能性（非vis absoluta）
4. 后台人通过操控前台人的意志决定来支配犯罪事实

与教唆犯的区分：
- 教唆犯：前台人仍保有自主决定能力
- 间接正犯：前台人已丧失意志自由
- 边界：前台人虽受强制但仍可选择 → 趋向教唆犯
```

#### 错误支配

```
典型情形：
1. 利用前台人的构成要件错误：
   - 前台人不知客观情况（如不知"糖水"实为毒药）
   - → 前台人无故意 → 后台人利用此错误支配犯罪
2. 利用前台人的禁止错误（学理处理；中国法对违法性认识无明文）：
   - 前台人不知行为违法
   - → 是否构成间接正犯？争议
3. 利用前台人的正当化情状错误：
   - 使前台人误以为存在正当防卫情状
   - → 前台人可能阻却责任 → 后台人利用此支配犯罪

后台人优势认知：
- 后台人知悉前台人不知的关键事实
- 正是这种"认知优势"构成了支配的基础
```

#### 组织支配

```
适用条件（组织支配通说判准）：
1. 具有权力层级的组织结构
2. 组织脱离法秩序
3. 执行者的可替代性
4. 组织领导者的下达命令

争议：
- 合法组织（如公司）内部是否适用？
- 少数说肯定（"企业间接正犯"），多数说否定或限制适用
```

### 间接正犯与教唆犯的界限

| 区分标准 | 间接正犯 | 教唆犯 |
|----------|----------|--------|
| 前台人地位 | 工具——缺乏自主决定 | 正犯——有完全的自主决定 |
| 后台人支配 | 有犯罪事实支配（学理） | 无支配，仅促成决意 |
| 前台人犯罪成立 | 不成立犯罪（至少欠缺某要素） | 成立犯罪（至少构成要件符合性+违法性） |
| 后台人的直接性 | 通过前台人"间接"实现 | 引起前台人"直接"实施 |

### "正犯后正犯"

- **问题**：前台人自身也成立正犯时，后台人是否仍可构成间接正犯？
- **通说**：否定——前台人为正犯时，后台人只能是教唆犯
- **例外**：组织支配情形下可能承认（前台人虽构成正犯，但组织支配仍使后台人为间接正犯）

## D. 教唆犯（《刑法》第 29 条）

> 第 29 条第 2 款"被教唆的人没有犯被教唆的罪，对于教唆犯，可以从轻或者减轻处罚"为中国法独立的教唆条款。教唆未满十八周岁的人犯罪的，应当从重处罚（第 29 条第 1 款）。

### 检视程式

```
前置确认：
  - 正犯不法行为已成立（至少构成要件符合性 + 违法性）—— 学理通说之限制从属性
  - 正犯至少达到未遂阶段（注：中国《刑法》第29条第2款另有"被教唆者未犯被教唆之罪"的独立教唆条款）

I. 构成要件符合性
   A. 客观构成要件
      1. 正犯不法行为的存在（至少已着手的不法行为）
      2. 教唆行为：
         a. 使他人产生犯罪决意
         b. 被教唆人原无犯罪决意（omnimodo facturus问题）
      3. 教唆行为与正犯犯罪决意之间的因果关系
   B. 主观构成要件——双重故意：
      1. 对教唆行为本身的故意：认识并意欲引起他人犯罪决意
      2. 对正犯行为及结果的故意：认识并意欲正犯实现构成要件

II. 违法性
III. 有责性
```

### 教唆行为的详解

#### 1. "使他人产生犯罪决意"

```
核心要求：
- 被教唆人之前无此犯罪的决意
- 教唆行为引起了被教唆人的犯罪决意
- 不要求被教唆人完全"无犯意"——对具体犯罪无决意即可

方式：
- 明确要求/请求/命令
- 暗示/引导/利用弱点
- 提供犯罪机会（争议——可能仅构成帮助）
- 不要求特定手段

排除教唆的情形——omnimodo facturus（已决定犯罪之人）：
- 被教唆人已有犯罪决意 → 教唆行为无因果力 → 不成立教唆
- 可能成立帮助犯（心理帮助/强化犯意）
```

#### 2. 因果关系

- 教唆行为与正犯犯罪决意之间须有因果关系
- "若无教唆，正犯不会产生犯罪决意"
- 部分教唆（升高犯罪决意程度）：争议——可能构成教唆或帮助

### 双重故意的内涵

| 故意层次 | 内容 | 程度 |
|----------|------|------|
| 第一层：对教唆行为 | 认识到自己的行为将引起他人犯罪决意 | 至少间接故意 |
| 第二层：对正犯行为 | 认识到正犯将实施特定犯罪并达到一定程度 | 至少间接故意 |

**教唆故意的特定性要求：**
- 须指向特定犯罪（不必精确到所有细节）
- 须指向特定人（不必确定唯一对象）
- 概括教唆（如"去犯点事"）→ 不符合特定性要求

## E. 从犯（辅助作用类型 / 帮助犯，《刑法》第 27 条）

> 第 27 条从犯包括"次要作用"与"辅助作用"两类，后者大致对应学理上的帮助犯；从犯应当从轻、减轻或者免除处罚。中国法"从犯"涵盖更广，与帮助犯并不完全等同。

### 检视程式

```
前置确认：
  - 正犯不法行为已成立（至少构成要件符合性 + 违法性）—— 学理通说之限制从属性
  - 正犯至少达到未遂阶段

I. 构成要件符合性
   A. 客观构成要件
      1. 正犯不法行为的存在
      2. 帮助行为：
         a. 物理帮助
         b. 心理帮助
      3. 帮助行为对正犯行为的因果贡献（促进关系）
   B. 主观构成要件——双重故意：
      1. 对帮助行为本身的故意
      2. 对正犯行为及其不法性质的故意

II. 违法性
III. 有责性
```

### 帮助行为的类型

| 类型 | 内涵 | 举例 |
|------|------|------|
| **物理帮助** | 提供工具、手段、信息等物质性协助 | 提供凶器、望风、运输赃物、提供场所 |
| **心理帮助** | 强化正犯犯意、提供技术建议、情感鼓励 | 鼓励、出谋划策、承诺事后帮助 |

### 帮助行为的因果性

```
争议焦点：帮助行为须达到何种因果贡献程度？

严格因果说：须证明"若无帮助，正犯不会犯罪/犯罪不会成功"
  → 要求过高，实践中难以证明

促进说（通说/本程式库采）：
  - 帮助行为"促进"了正犯行为即可
  - 不要求"条件关系"意义上的因果
  - 使正犯犯罪更容易、更安全、更有效即满足
  - 即便没有帮助正犯也会犯罪，但帮助使犯罪变得更便利

具体表现：
  - 物理帮助：使犯罪实施在物理层面更加便利
  - 心理帮助：使正犯的犯罪决意更加坚定
  - 注意：单纯在场且不反对 ≠ 心理帮助（除非有保证人义务）
```

### 帮助的时间

| 时间段 | 可否构成帮助 | 说明 |
|--------|-------------|------|
| 预备阶段 | 可以 | 事前提供工具、信息等 |
| 实行阶段 | 可以 | 最典型的帮助形态 |
| 既遂后 | 原则不可 | 除非事前承诺事后帮助（事前的心理帮助） |

## 专题：共犯过剩/实行过限

### 概念

正犯的实际行为超出了共同犯罪计划的范围。

### 处理规则

```
基本原则：
  - 各参与者仅对共同犯罪计划范围内的行为与结果负责
  - 超出计划的部分（过剩部分）仅由实际实施者自负

判断步骤：
1. 确定共同犯罪计划的范围（内容+程度）
2. 比较正犯实际行为与计划
3. 超出部分：其他参与者不负责任

类型：
┌──────────────────────────────────────┐
│ 量的过剩                              │
│ - 同一类犯罪但程度加重              │
│ - 例：计划伤害，实际杀人            │
│ → 共犯仅对计划范围（伤害）负责       │
├──────────────────────────────────────┤
│ 质的过剩                              │
│ - 实施完全不同种类的犯罪            │
│ - 例：计划盗窃，实际强奸            │
│ → 共犯对过剩犯罪完全不负责          │
└──────────────────────────────────────┘
```

## 专题：部分犯罪共同说

### 适用场景

共同犯罪人之间存在不同罪名的故意时，如何认定共同犯罪？

### 处理规则

```
规则：
  - 在各参与者故意"重叠"的范围内，成立共同犯罪
  - 重叠部分 = 各人故意的交集

例：甲有杀人故意，乙仅有伤害故意，二人共同对丙施暴
  → 在"故意伤害"范围内成立共同犯罪
  → 甲另就"故意杀人"单独负责
  → 乙不对死亡结果承担故意杀人的共同正犯责任
    （可能承担故意伤害致死的结果加重犯责任）
```

## 专题：共犯脱离

### 概念

共犯（或共同正犯）在犯罪既遂前退出共同犯罪。

### 脱离的条件

| 参与形态 | 脱离要求 | 效果 |
|----------|----------|------|
| 共同正犯 | 积极消除自己贡献的因果力 + 其他共犯知悉 | 对脱离后的行为不负责 |
| 教唆犯 | 消除被教唆人的犯罪决意 / 有效阻止犯罪 | 不成立教唆犯 |
| 帮助犯 | 撤回帮助行为的实际效果 | 不成立帮助犯 |

### 脱离的效果

```
1. 成功脱离 → 对脱离后发生的行为与结果不负刑事责任
2. 仅表示退出但未实际消除影响 → 脱离不成立
3. 脱离前已完成的行为 → 仍可能构成预备犯或独立犯罪
4. 与中止犯的关系：
   - 共犯脱离 ≈ 共犯的"中止"
   - 但脱离的标准可能低于中止（不要求自动性）
```

## 专题：身份犯中的共同犯罪

### 问题

特殊身份是仅要求正犯具备，还是也要求共犯具备？

### 处理规则

```
真正身份犯：
  - 身份 = 构成要件要素
  - 无身份者不能构成正犯
  - 无身份者可构成共犯（教唆/帮助）—— 学理通说之限制从属性的结论
  - 理由：共犯借用正犯的不法

不真正身份犯：
  - 身份 = 加重/减轻要素
  - 有身份者适用加重/减轻条款
  - 无身份者适用普通条款
  - 共同犯罪中：分别适用（通说）

中国法处理（《刑法》第29条+司法解释）：
  - 基本遵循上述原则
  - 实践中：以利用职务便利之人的身份定罪
```

## 专题：共同犯罪中的认识错误

### 一、正犯的错误对共犯的影响

| 正犯错误类型 | 对共犯的影响 |
|-------------|-------------|
| 正犯的对象错误 | 共犯对实际被害人负责（对象等价） |
| 正犯的打击错误 | 争议（同正犯独立犯罪时的打击错误处理） |
| 正犯的因果偏离 | 在一般可预见范围内，共犯仍负责 |

### 二、共犯自身的错误

| 错误情形 | 处理 |
|----------|------|
| 教唆犯误认对象 | 不影响教唆故意（对象等价） |
| 教唆犯对正犯行为范围认识错误 | 仅在共同故意范围内负责 |
| 帮助犯不知正犯将犯何种具体犯罪 | 须有对正犯犯罪类型的概括认识 |
