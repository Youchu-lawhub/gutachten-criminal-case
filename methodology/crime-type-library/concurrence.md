# 罪数与竞合

> **体系定位**：罪数与竞合理论处于犯罪论的最后阶段——当确认行为人的行为构成犯罪后，需要判断成立几个犯罪以及如何处罚。本理论解决的核心问题是：如何对行为人的全部犯罪行为进行**整体评价**与**科刑**。

---

## 一、罪数论的体系结构

### 1.1 总体框架

```
罪数判断流程：

Step 1: 确定"行为单数"还是"行为复数"
         （一个行为 or 数个行为？）
              │
        ┌─────┴─────┐
        ▼           ▼
   行为单数      行为复数
        │           │
        ▼           ▼
Step 2: 触犯一个   触犯一个
   还是数个罪名？  还是数个罪名？
        │           │
   ┌────┴────┐      ▼
   ▼         ▼   实质竞合
 一个罪名  数个罪名  （数罪并罚）
   │         │
   ▼         ▼
 单纯一罪  检验关系：
           法条竞合 or 想象竞合？
```

### 1.2 核心概念对照

| 概念 | 行为数 | 罪名数 | 处罚方式 |
|------|--------|--------|----------|
| 单纯一罪 | 一 | 一 | 单罪处罚 |
| 法条竞合 | 一 | 形式上数/实质一 | 仅适用一条 |
| 想象竞合 | 一 | 数 | 从一重处断 |
| 实质竞合 | 数 | 数 | 数罪并罚 |

## 二、行为单数与行为复数的判断

### 2.1 判断的重要性

行为单数/复数的判断是竞合论的**前置问题**：
- 行为单数 → 可能为法条竞合或想象竞合（从一重处断）
- 行为复数 → 实质竞合（数罪并罚）

### 2.2 行为单数

#### 自然行为单数

```
概念：从自然观察的角度看，构成一个统一行为的情形

判断标准：
1. 时空紧密性：
   - 在同一时间、同一地点实施
   - 各行为之间无时间间断

2. 统一意思决定：
   - 基于同一个犯罪决意
   - 非经过新的意思决定

3. 外观上的统一性：
   - 客观上看起来是"一件事"
   - 非明显分离的数个事件

典型情形：
- 连续数拳打击同一被害人 → 一个行为
- 一枪射穿两人 → 一个行为
- 在同一入室盗窃中拿取多件物品 → 一个行为
```

#### 法律行为单数

即便从自然观察角度看可能是数个行为，但法律将其评价为一个行为：

##### (1) 继续犯

```
概念：犯罪行为创设的违法状态在一定时间内持续存在

特征：
- 开始行为 → 持续状态 → 终了
- 整个持续过程视为"一个行为"

典型罪名：
- 非法拘禁（持续剥夺自由）
- 非法持有枪支（持续占有）
- 窝藏犯（持续窝藏）

法律意义：
- 追诉时效从终了时起算
- 持续期间内实施的其他犯罪与继续犯构成想象竞合（括弧效应）
```

##### (2) 连续犯——已被废弃

```
传统概念：
  基于概括故意，连续实施同一犯罪行为
  → 视为一个犯罪

现状（中国法）：
  - 理论上已趋于废弃
  - 实务中仍有适用但日趋限缩
  - 同种数罪原则上应数罪并罚
```

##### (3) 结合犯

```
概念：法律将数个独立犯罪行为结合为一个独立罪名

例：
- 抢劫罪 = 暴力/胁迫 + 取财（结合了强制罪与盗窃罪的要素）
- 入户盗窃 = 非法侵入住宅 + 盗窃

法律效果：
- 被结合的各行为不再独立评价
- 适用结合后的罪名与法定刑
- 不存在竞合问题（已被立法整合）
```

##### (4) 集合犯

```
概念：构成要件本身预设了行为的反复实施

类型：
- 营业犯：以营利为目的反复实施
  - 例：走私罪（多次走私构成一罪）
- 常业犯：以犯罪为常业
  - 例：已废除的"常业诈骗"

法律效果：
- 多次行为评价为一罪
- 但注意：中国法中多数"多次犯罪"规定为加重情节而非独立罪名
```

### 2.3 行为复数

```
判断标准（反面）：
- 不符合自然行为单数的条件
- 不符合法律行为单数的条件
→ 则为行为复数

典型情形：
- 不同时间对不同被害人分别实施侵害
- 经过新的犯罪决意后再次犯罪
- 时间间隔较长的两次犯罪行为
```

### 2.4 行为单复数的判断标准总结

| 判断因素 | 倾向行为单数 | 倾向行为复数 |
|----------|-------------|-------------|
| 时间跨度 | 短暂/无间断 | 较长间隔 |
| 同一被害人 | 是 | 否（不同被害人） |
| 同一地点 | 是 | 否（不同地点） |
| 统一犯意 | 同一犯罪决意 | 新的犯罪决意 |
| 行为外观 | 自然观察为一体 | 明显分离的事件 |

## 三、法条竞合

### 3.1 概念

一个行为表面上符合数个刑法条文的构成要件，但由于条文之间的逻辑关系，实质上仅有一个条文应予适用。

### 3.2 与想象竞合的区分

| 比较项 | 法条竞合 | 想象竞合 |
|--------|----------|----------|
| 本质 | 条文间的逻辑包含关系 | 一行为实质触犯数独立罪名 |
| 判断方法 | 从法条关系入手（抽象判断） | 从案件事实入手（具体判断） |
| 不法评价 | 适用的一条已完整评价全部不法 | 单一条文不能完整评价 |
| 处罚 | 仅适用一条（排斥其余） | 从一重处断（但不得低于轻罪最低刑） |
| 被排斥条文 | 完全不适用（不出现在判决中） | 仍在判决中列明（仅不单独科刑） |

### 3.3 法条竞合的类型

#### (1) 特别关系

```
概念：
  甲条文的构成要件完全包含乙条文的构成要件，且附加了额外要素
  → 特别法优于一般法（lex specialis derogat legi generali）

逻辑关系：
  特别条文 = 一般条文 + 特别要素

判断方法：
  A条构成要件要素 ⊃ B条构成要件要素
  → A为特别法，适用A，排斥B

典型实例：
┌──────────────────────────────────────────────┐
│ 特别法          一般法                        │
│ 诈骗罪          盗窃罪（×不适用）             │
│   → 因为诈骗另有"欺骗手段"要素              │
│                                              │
│ 抢劫罪          盗窃罪 + 故意伤害罪           │
│   → 抢劫 = 暴力/胁迫 + 取财                 │
│                                              │
│ 故意杀人        故意伤害                      │
│   → 杀人包含了伤害（伤害是杀人的必经阶段）   │
│                                              │
│ 贪污罪          职务侵占罪                    │
│   → 贪污 = 职务侵占 + 国家工作人员身份      │
└──────────────────────────────────────────────┘
```

#### (2) 补充关系

```
概念：
  某条文明示或默示仅在其他条文不适用时方可适用
  → 主要法优于补充法（lex primaria derogat legi subsidiariae）

类型：
1. 明示补充：
   - 条文明确规定"本法另有规定的，依照规定"
   - 例：第233条过失致人死亡"本法另有规定的，依照规定"

2. 默示补充：
   - 基于条文体系关系推断
   - 较轻罪为较重罪的补充
   - 未遂为既遂的补充
   - 预备为着手的补充
   - 危险犯为实害犯的补充

典型实例：
- 过失致人死亡罪 vs 交通肇事罪：后者为特别规定，排斥前者
- 故意伤害（轻伤）vs 故意伤害（重伤）：重伤排斥轻伤
- 未遂 vs 既遂：既遂排斥未遂
```

#### (3) 吸收关系

```
概念：
  一个犯罪在典型情况下必然伴随另一个犯罪
  → 主行为吸收从行为（被吸收行为不再独立评价）

判断标准：
  - 非逻辑上的必然包含（区别于特别关系）
  - 而是经验上的"典型伴随"
  - 较重犯罪吸收较轻犯罪

典型实例：
┌──────────────────────────────────────────────┐
│ 吸收行为          被吸收行为                  │
│                                              │
│ 入室盗窃          非法侵入住宅              │
│   → 入室是盗窃的手段                        │
│                                              │
│ 故意杀人          故意伤害                    │
│   → 杀人过程中的伤害被吸收                  │
│                                              │
│ 抢劫              非法拘禁（短暂）           │
│   → 抢劫中短暂限制自由被吸收               │
└──────────────────────────────────────────────┘

注意：
- 吸收关系适用范围应严格限缩
- 不能以"重罪吸收轻罪"为由随意适用
- 须为"典型的""经验上必然伴随的"行为
```

### 3.4 法条竞合的实质判断标准

```
核心问题：仅适用一条是否已"完整评价"了全部不法内容？

是 → 法条竞合（仅适用一条即可）
否 → 想象竞合（须列明数条并从一重处断）

判断方法：
1. 适用的单一条文是否已涵盖全部行为无价值？
2. 适用的单一条文是否已涵盖全部结果无价值？
3. 是否存在"剩余不法"未被评价？
4. 是否涉及不同法益主体？

→ 涉及不同法益主体时，通常应为想象竞合（非法条竞合）
   例：一枪射穿甲乙二人 → 对甲和乙分别成罪 → 想象竞合
```

## 四、想象竞合（《刑法》第69条解释适用）

### 4.1 概念

一个行为（行为单数）同时满足数个不同犯罪构成要件，且各罪名之间不存在法条竞合关系。

### 4.2 成立要件

```
想象竞合 = 行为单数 + 触犯数个独立罪名 + 非法条竞合

要件：
1. 行为单数：从自然/法律角度看为一个行为
2. 触犯数个罪名：该行为满足数个犯罪构成要件
3. 各罪名独立：不存在特别、补充、吸收关系
4. 涉及不同面向的不法评价：单一罪名不能完整评价全部不法
```

### 4.3 典型情形

| 情形 | 涉及罪名 | 想象竞合理由 |
|------|----------|-------------|
| 一枪射穿甲乙 | 杀甲（既遂）+ 伤乙 | 一个射击行为侵害两人法益 |
| 醉驾撞死行人 | 危险驾驶 + 交通肇事 | 同一驾驶行为触犯两罪 |
| 强奸过程中致伤 | 强奸 + 故意伤害 | 一个暴力行为的双重评价（如超出强奸通常暴力） |
| 公务员受贿后枉法裁判 | 受贿罪 + 枉法裁判罪 | 争议——可能为牵连犯/想象竞合 |

### 4.4 法律效果

```
处罚原则：从一重处断

具体适用：
1. 确定各罪的法定刑
2. 选择最重的法定刑（"重罪"）作为处断刑
3. 限制：不得低于各轻罪中最高的最低法定刑
   → "从一重处断但不得低于轻罪最低刑"

例：
  A罪法定刑：3-10年
  B罪法定刑：5-15年
  → 适用B罪法定刑（5-15年）
  → 实际量刑不得低于5年（B罪最低刑）

在判决中的表述：
  "被告人犯A罪、B罪，属想象竞合，依照处罚较重的B罪定罪处罚"
  → 所有罪名均列明，但仅从一重科刑
```

### 4.5 想象竞合的"明示机能"

```
功能：即便不单独科刑，列明全部罪名仍有重要意义：

1. 完整评价不法：表明行为的全部不法面向均已被刑法评价
2. 前科记录：全部罪名记入前科
3. 量刑考量：轻罪的成立可作为从重量刑因素
4. 共犯定性：对共犯的处理可能因罪名不同而异
5. 附随效果：某些罪名可能触发特殊附随效果
```

## 五、实质竞合（《刑法》第69条）

### 5.1 概念

行为人以数个独立行为（行为复数），实施了数个独立犯罪。

### 5.2 成立要件

```
实质竞合 = 行为复数 + 各行为分别构成独立犯罪

要件：
1. 行为复数：存在两个或以上独立行为
2. 各行为分别满足不同犯罪构成要件（异种数罪）
   或分别满足同一犯罪构成要件（同种数罪）
3. 各行为之间不存在将其统一评价为"一个行为"的法律基础
```

### 5.3 法律效果——数罪并罚

```
《刑法》第69条规定的数罪并罚原则：

1. 限制加重原则：
   - 数罪中有一罪被判无期 → 执行无期
   - 数罪中有一罪被判死刑 → 执行死刑
   - 数罪均为有期自由刑：
     → 在总和刑以下、最高刑以上酌情决定执行刑期
     → 但不得超过总和刑

2. 附加刑的并罚：
   - 罚金：可合并执行或分别执行
   - 剥夺政治权利：可附加执行
```

### 5.4 同种数罪与异种数罪

| 类型 | 含义 | 处理 |
|------|------|------|
| 异种数罪 | 数行为构成不同罪名 | 数罪并罚（无争议） |
| 同种数罪 | 数行为构成相同罪名 | 数罪并罚（通说）/ 部分判例作为量刑情节 |

## 六、不罚之前后行为

### 6.1 不罚的前行为

```
概念：
  某一犯罪行为之前实施的行为，其不法已被后续主行为"吸收评价"

判断标准：
  - 前行为针对同一法益
  - 前行为的不法程度低于主行为
  - 主行为对前行为的不法已完整评价
  - 前行为为主行为的"典型前阶段"

典型情形：
  - 盗窃前的非法侵入 → 被盗窃吸收
  - 伪造货币前的准备工具 → 被伪造行为吸收
  - 诈骗前的伪造文书 → 视具体情况（争议较大）
```

### 6.2 不罚的后行为

```
概念：
  犯罪既遂后实施的行为，系对犯罪成果的"确保"，
  不构成新的法益侵害或不法程度未超出前罪

判断标准：
  - 后行为针对同一法益（同一被害人）
  - 后行为未侵害新的法益或法益主体
  - 后行为系前罪成果的"自然延伸"
  - 后行为的不法已被前罪的处罚所涵盖

典型情形：
  - 盗窃后对赃物的使用/消费 → 不罚的后行为
  - 盗窃后的销赃 → 争议（如涉及第三人可能独立评价）
  - 故意毁坏财物后丢弃残余物 → 不罚的后行为

例外——独立评价的后行为：
  - 后行为侵害了新的法益/新的被害人
  - 后行为的不法程度超出前罪涵盖范围
  - 例：盗窃后销毁证据（妨害司法） → 可能独立评价
```

## 七、继续犯的"括弧效应"

### 7.1 概念

继续犯的持续过程将其间实施的其他犯罪"括弧"在一起，使原本为实质竞合的数罪通过继续犯的统一行为性质而构成想象竞合。

### 7.2 机制

```
原理：
  继续犯的整个持续过程 = 一个行为（法律行为单数）
  → 在此期间实施的其他犯罪 = 与继续犯"同时"发生
  → 一个行为（继续犯）+ 数个罪名 = 想象竞合

图示：
  非法拘禁（持续中）: ============================
  期间犯罪A:              |||
  期间犯罪B:                         |||
  
  若无继续犯：A与B为实质竞合（数罪并罚）
  有继续犯括弧：非法拘禁 + A → 想象竞合
               非法拘禁 + B → 想象竞合
               → 整体处理为想象竞合？

限制：
  - 括弧效应不能无限延伸
  - 被括弧的犯罪须与继续犯有关联性
  - 完全偶然的时间重叠不应适用括弧效应
```

### 7.3 争议

| 立场 | 主张 | 理由 |
|------|------|------|
| 肯定说 | 括弧效应使期间犯罪统一为想象竞合 | 继续犯为一行为，期间犯罪与之"竞合" |
| 限制说 | 仅当期间犯罪与继续犯有内在关联时适用 | 避免不合理的处罚减轻 |
| 否定说 | 不承认括弧效应 | 继续犯的"一行为"特征不应影响独立犯罪的评价 |

## 八、处理顺序

### 8.1 完整的竞合处理流程

```
Step 1: 确定各独立犯罪是否成立
  → 对每一候选罪名完整进行三阶层检验

Step 2: 确定行为单数/复数
  → 判断各犯罪行为之间的关系
  → 自然行为单数？法律行为单数？行为复数？

Step 3: 行为单数情形——区分法条竞合与想象竞合
  → 条文间有特别/补充/吸收关系？
  → 是 → 法条竞合（仅适用一条）
  → 否 → 想象竞合（从一重处断）

Step 4: 行为复数情形——实质竞合
  → 数罪并罚

Step 5: 检查不罚的前后行为
  → 是否有应被吸收不再独立评价的前行为/后行为？

Step 6: 最终结论
  → 综合以上分析，得出最终罪名与处罚结论
```

### 8.2 竞合结论的表述模式

```
表述模板：

法条竞合：
  "甲的行为构成A罪，A罪与B罪为法条竞合（特别关系），
   仅以A罪论处。"

想象竞合：
  "甲的一行为同时构成A罪与B罪，属想象竞合，
   依照处罚较重的A罪定罪处罚。"

实质竞合：
  "甲先后实施的行为分别构成A罪与B罪，
   应当数罪并罚。"
```

## 九、特殊竞合形态

### 9.1 牵连犯——争议形态

```
传统概念：
  犯罪的手段行为或结果行为另触犯其他罪名
  - 手段牵连：手段行为本身构成独立犯罪
  - 结果牵连：犯罪结果引起另一犯罪

中国法现状：
  - 传统通说承认牵连犯为独立竞合形态
  - 处理：从一重处断
  - 趋势：逐步被想象竞合或实质竞合取代
  - 新观点：牵连犯概念应当废弃
    → 手段牵连 = 可能的想象竞合（如一行为）或实质竞合（如数行为）
    → 结果牵连 = 不罚的后行为或实质竞合

本程式库立场：
  原则不采用牵连犯概念，回归行为单复数判断 + 标准竞合规则。
  但标注传统牵连犯理论的处理结论。
```

### 9.2 吸收犯——边界模糊

```
传统概念：
  一犯罪行为吸收另一犯罪行为，仅以一罪论

与法条竞合中"吸收关系"的重叠：
  - 本程式库将"吸收"作为法条竞合的一种类型处理
  - 不单独设立"吸收犯"范畴
```

## 十、同一行为人多事实单元的整体处理

### 10.1 处理原则

```
整体处理流程：

1. 将案件事实分割为若干"事实单元"
2. 在每一事实单元内：
   a. 识别候选罪名
   b. 完成三阶层检验
   c. 确定该单元内的竞合关系
3. 各事实单元之间：
   a. 判断行为单复数
   b. 确定竞合类型
4. 最终整合为完整的竞合结论
```

### 10.2 多被害人情形

```
原则：不同被害人的法益不能相互吸收

规则：
  - 侵害不同被害人 → 原则上各自成罪
  - 一行为侵害多人 → 想象竞合（每一被害人一个罪名）
  - 数行为侵害多人 → 实质竞合

例：
  甲一枪射穿A（死亡）和B（重伤）
  → 对A：故意杀人既遂
  → 对B：故意伤害（或过失致人重伤）
  → 一个行为 → 想象竞合 → 从一重处断
```
