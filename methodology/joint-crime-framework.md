# 共同犯罪分析框架

> **核心命题**：当同一犯罪事实涉及多名参与人时，对共同犯罪关系的分析不应在每位参与人章节中重复展开。本框架提供标准化的"共犯关系基线"机制，使首次分析建立完整论证，后续参与人通过交叉引用复用基线结论，从而消除冗余论证、保持逻辑一致性。
>
> **适用范围**：同一事实单元内存在两名以上参与人、且至少一对参与人之间成立共同犯罪关系的情形。单独正犯案件无需调用本框架。

---

## 一、四步分析模板

### 第一步：共同犯罪关系确认

在进入个别参与人的三阶层检视之前，先确认各参与人之间是否成立共同犯罪关系。

| 检验维度 | 内容 | 判断 |

| 联络形式 | 明示 / 默示 / 暗示 | [填入] |
| 形成时点 | 预备阶段 / 实行阶段 | [填入] |
| 联络内容 | 对犯罪目标、分工的共同认识 | [填入] |

#### 1.2 行为分工

| 参与人 | 分工角色 | 核心行为 | 贡献性质 |
|--------|----------|----------|----------|
| [A] | [描述] | [描述] | 实行 / 教唆 / 帮助 |
| [B] | [描述] | [描述] | 实行 / 教唆 / 帮助 |
| [C] | [描述] | [描述] | 实行 / 教唆 / 帮助 |

#### 1.3 结论

确认共同犯罪关系成立的范围：哪些参与人之间、就哪些犯罪行为成立共同犯罪。若存在部分犯罪共同说之适用（各参与人故意不完全一致），须标注重叠范围。

---

### 第二步：参与形态确定

基于第一步确认的共同犯罪关系，按犯罪事实支配理论（[参见 `crime-type-library/joint-crime.md` §A.三](crime-type-library/joint-crime.md)）确定各参与人的参与形态：

| 参与形态 | 判定标准 | 适用参与人 |
|----------|----------|------------|
| **正犯（直接实行）** | 亲自实施构成要件行为，具有行为支配 | [填入] |
| **正犯（间接正犯）** | 通过意思支配利用他人作为工具 | [填入] |
| **正犯（共同正犯）** | 基于共同计划，功能性支配，"部分实行全部责任" | [填入] |
| **教唆犯** | 引起原无犯意之人的犯罪决意（《刑法》第29条） | [填入] |
| **帮助犯** | 对正犯行为提供物理或心理促进（《刑法》第27条"辅助作用"） | [填入] |

详细判定标准参见 [`crime-type-library/joint-crime.md`](crime-type-library/joint-crime.md) 各节。

---

### 第三步：主从犯确定

参与形态确定后，依《刑法》第26-29条的法定分类确定各参与人的主从犯地位（[参见 `06-general-framework.md` §5-§8](offense-examination-engine/06-general-framework.md)）：

| 类别 | 法定标准 | 法律后果 |
|------|----------|----------|
| **主犯**（第26条） | 组织、领导犯罪集团进行犯罪活动，或在共同犯罪中起主要作用 | 按其所参与或组织、指挥的全部犯罪处罚 |
| **从犯**（第27条） | 在共同犯罪中起次要作用或辅助作用 | **应当**从轻、减轻或免除处罚 |
| **胁从犯**（第28条） | 被胁迫参加犯罪 | **应当**减轻或免除处罚 |
| **教唆犯**（第29条） | 教唆他人犯罪 | 按其在共同犯罪中所起的作用处罚 |

注意：参与形态（正犯/共犯）与主从犯（法定分类）并非一一对应。正犯通常为但不必然是主犯；帮助犯通常为但不必然是从犯。须分别论证。

---

### 第四步：交叉引用机制

#### 4.1 基线建立

首位被分析的参与人（通常为在共同犯罪中起最核心作用者）负责建立"共犯关系基线"。基线包含：

- 第一步至第三步的完整论证
- 共同犯罪关系的整体判断
- 各参与人的参与形态与主从犯初步定位

#### 4.2 引用格式

后续参与人在涉及同一共同犯罪事实时，不再重复第一步至第三步的完整论证，而以交叉引用替代：

```
共同犯罪关系详见[首次分析参与人]§[罪名]基线分析。

本案事实单元U[X]中，[当前参与人]与[其他参与人]之间的共同犯罪
关系、意思联络、行为分工及参与形态定性，已在[首次分析参与人]
§[罪名]§共犯关系基线中完整论证，此处不再赘述。

基于基线分析，[当前参与人]的参与形态为[参与形态]，主从犯地位
为[主犯/从犯/胁从犯]。以下仅就[当前参与人]的个别要件（违法性、
有责性等）展开独立检验。
```

#### 4.3 例外：须独立论证的情形

以下情形不适用交叉引用，须独立完整论证：

- 当前参与人的参与形态或主从犯地位与基线分析存在分歧
- 当前参与人涉及共犯过剩（实行过限）问题
- 当前参与人涉及共犯脱离
- 当前参与人的违法性或有责性判断与共同犯罪关系密切关联

## 三、与 W3 并行写作流程的衔接

### 3.1 触发条件

当 W2 大纲（[参见 `05-outline-assembly.md`](offense-examination-engine/05-outline-assembly.md)）中同一事实单元标记了两名以上参与人的同一或关联罪名时，W3 编排器（[参见 `3-drafting.md`](../workflow/3-drafting.md)）应当：

1. 识别该事实单元中的"基线参与人"——通常为实行行为的核心实施者或主犯
2. 指定负责基线参与人的 subagent 承担基线建立任务
3. 在任务指令中明确要求：完成共犯关系基线后，将基线结论写入共享上下文

### 3.2 执行协议

```
基线 subagent（负责参与人A）：
  → 完整执行第一步至第三步
  → 输出共犯关系基线（含基线表格 + 分析要点）
  → 基线写入共享上下文，标记为 [基线-已就绪]

引用 subagent（负责参与人B/C/...）：
  → 检测共享上下文中的 [基线-已就绪] 标记
  → 在参与人章节中插入交叉引用段落（§一、第四步格式）
  → 仅展开当前参与人的个别要件独立检验
```

### 3.3 效率提升

在典型的多人共同犯罪案件中，共同犯罪关系论证（意思联络、行为分工、参与形态、主从犯）通常占每位参与人章节的 25%-35% 篇幅。通过基线机制：

- 基线参与人：完整论证（篇幅不变）
- 其余参与人：以交叉引用替代重复论证，仅保留个别差异分析

**预估效果**：对于涉及 3 名以上参与人的复杂共同犯罪案件，可减少约 40% 的冗余论证量，同时确保各参与人章节的逻辑一致性。

### 3.4 与比例原则的协调

基线机制不改变比例原则（[参见 `proportionality.md`](proportionality.md)）的适用。基线参与人的共犯关系论证按标准深度展开；引用参与人如需对基线结论提出异议或补充，按第三层（深度鉴定式）独立展开。

---

## 四、自查清单

```
□ 是否在大纲中识别了需要基线机制的共同犯罪事实？
□ 基线参与人是否为首要正犯或核心主犯？
□ 基线是否包含完整的四步分析（关系确认→参与形态→主从犯→引用格式）？
□ 后续参与人是否正确插入交叉引用段落？
□ 是否存在不适用交叉引用的例外情形（过限/脱离/分歧）？
□ 基线与比例原则是否协调（争议问题是否获得充分深度）？
□ 各参与人的违法性、有责性是否仍为独立检验？
```
