# 内在结构：三阶层检验逻辑与涵摄技术

> 本检视框架基于中华人民共和国现行《刑法》（2023 修正）。中国法未明文规定的事项（违法性认识、容许构成要件错误、不能犯、限制从属性等）属学理解释。

> **🔀 学派立场配置声明（中立—可配置—默认行为无价值二元论）**
>
> 鉴于中国当代刑法学在犯罪论体系内部学派分歧显著，本技能**不预设任何一种学说为权威**。本技能识别中国当代刑法学在三阶层框架内部存在的**两条主要解释主轴**：
>
> 1. **行为无价值二元论**（下称"立场 A"）——主张违法性同时包含行为无价值（违反规范的行为方式）与结果无价值（法益侵害）。在偶然防卫、正当化主观要素、不能犯、未遂犯成立等问题上采较严格立场。
> 2. **结果无价值论**（下称"立场 B"）——主张违法性的本质仅在于法益侵害（结果无价值），行为无价值只是认定结果无价值的征表。在偶然防卫、正当防卫主观要素、§20 第 3 款性质、客观归责本土化、被害人承诺、紧急避险等问题上与立场 A 形成明显分歧。
>
> **配置入口**：用户可在 `workflow/0-cold-start-interview.md` 段落 1.5 对 3 项重大分歧议题各自选择立场，写入 `.user-config.md` 的 `doctrinal_stances` 字段。**若用户未做任何选择，本技能默认采立场 A（行为无价值二元论）**。其余体系性分歧（客观归责本土化、§20第3款性质、容许构成要件错误、不能犯/未遂区分等）由本技能按推荐立场自行处理，无需用户配置。
>
> **agent 写作规则**：
>
> - 默认按用户配置或本技能 default（立场 A）展开论证；
> - 当案件事实落在异见说有重要差异的情境（典型如：偶然防卫、假想防卫、防卫挑拨、容许构成要件错误、客观归责的具体应用、§20 第 3 款的扩张/限缩、抢劫罪复行为、注意规定 vs 法律拟制识别）时，**必须显性展开学派分歧**并最终选边站；
> - 在分则解释问题上，**默认调用** `interpretation-toolkit.md` 下编——该工具箱已按双轨呈现机制内置可切换立场。
> - **总则采立场 A、分则解释吸收立场 B 的部分主张**是一种被广泛接受的折衷路径，agent 在跨议题切换时须显性声明并给出理由。

## 一、三阶层递进检验逻辑

### 1.1 基本体系架构

本技能采用**三阶层犯罪论体系**，以**行为无价值二元论**为理论基础：

| 阶层 | 检验方向 | 功能定位 |
|------|----------|----------|
| 第一阶层：构成要件符合性 | 正向积极判断 | 类型化不法行为 |
| 第二阶层：违法性 | 反向消极判断 | 排除正当化事由 |
| 第三阶层：有责性 | 反向消极判断 | 确认个人可非难性 |

### 1.2 阶层间的逻辑关系

```
构成要件符合性(+) ──→ 违法性推定成立（不法征表机能）
                        │
                        ├─ 无正当化事由 → 违法性(+)
                        │                    │
                        │                    └─ 有责性检验
                        │                         ├─ 无免责事由 → 有责性(+) → 犯罪成立
                        │                         └─ 有免责事由 → 有责性(-) → 犯罪不成立
                        │
                        └─ 有正当化事由 → 违法性(-) → 犯罪不成立（无需进入有责性）
```

**核心原理**：

- **构成要件的不法征表机能**：构成要件符合性的肯定，创设了行为违法的推定。这一推定可被正当化事由推翻。
- **阶层消极判断的终止效力**：任何一个阶层得出否定结论，即停止对该罪名的继续检验。
- **不可逆序原则**：不得在未确认构成要件符合性的情况下直接讨论违法性或有责性。

### 1.3 阶层递进的实践意义

> **检验纪律**：每一阶层必须在前一阶层得出肯定结论后方可进入。否定结论意味着对该罪名的全部检验终止，直接转入下一个待检罪名。

阶层递进的意义在于：

1. **避免无效论证**：若构成要件不符合，则无需讨论正当防卫等问题
2. **确保逻辑清晰**：读者可以清楚知道每一步论证的功能定位
3. **防止要素混淆**：将不同功能的要素分配到各自阶层，避免"体系内流浪"

## 二、涵摄四步法

### 2.1 涵摄的基本含义

涵摄是将具体案件事实归入抽象法律规范的逻辑操作。其本质是一个**三段论**：

```
大前提：法律规范及其构成要件要素的定义
小前提：案件事实
结论：案件事实是否符合法律规范
```

### 2.2 四步操作规程

#### 第一步：总起句

**功能**：提出待检验的假设，划定本段论证的范围。

**格式模板**：

```
[行为人]的[具体行为]可能构成[法条编号][罪名]中的[具体要素]。
```

**示例**：

```
甲将乙推入河中的行为可能符合刑法第232条故意杀人罪中的"杀害他人"这一构成要件行为要素。
```

**注意事项**：
- 总起句必须是假设性的（使用"可能"、"或许"等词）
- 总起句的范围应精确对应一个具体要素
- 结论句必须在形式上与总起句对应

#### 第二步：定义

**功能**：明确法律概念的内涵与外延，为归入提供评判标准。

**核心技术——局部定义法**：

> 不必给出某一概念的完整定义，只需给出与本案事实匹配的那一层面的定义。这样可以避免不必要的学说争议，同时确保定义的精确性。

**定义来源层级**：

| 优先级 | 来源 | 说明 |
|--------|------|------|
| 1 | 法律明文定义 | 刑法条文中的立法定义（如第14条对"故意"的定义） |
| 2 | 司法解释定义 | 最高人民法院/检察院的司法解释中的概念界定 |
| 3 | 通说学理定义 | 刑法教科书中的通行定义 |
| 4 | 判例中的操作定义 | 指导性案例或公报案例中采用的定义 |
| 5 | 学说争议下的择一定义 | 在学说展开段落中选定的立场所采用的定义 |

**示例**：

```
所谓"杀害他人"，是指以足以引发死亡结果的方式对他人生命法益实施侵害的行为
（局部定义：此处仅从行为方式角度定义，因本案的争点在于行为手段的致命性）。
```

#### 第三步：归入

**功能**：将案件事实与定义进行比对，判断事实是否落入定义的涵摄范围。

**核心技术——层层追问法**：

```
定义要素A → 事实中有无对应？ → 若有，是否完全满足？
定义要素B → 事实中有无对应？ → 若有，是否完全满足？
……
所有要素均满足 → 归入成功
任一要素不满足 → 归入失败
```

**归入的语言标志**：

| 适合 | 不适合 |
|------|--------|
| "此处/本案中……" | "我认为……" |
| "根据案件事实……" | "显然……" |
| "甲的行为在客观上……" | "毫无疑问……" |

**示例**：

```
此处，甲将乙推入深度超过3米的河流中，而乙不会游泳。将不会游泳者推入深水，
在客观上具有足以引发溺亡结果的高度危险性，属于以足以引发死亡结果的方式
实施的侵害行为。
```

**嵌套追问的运用**：

当某一事实要素是否满足定义存在疑问时，需要进一步追问：
- "何为'足以引发死亡结果的方式'？"
- "3米深的水域对不会游泳者是否构成足以致命的环境？"
- 逐层追问至能够直接从事实得出是/否判断为止

#### 第四步：结论

**功能**：对本轮涵摄给出明确的中间结论。

**形式要求**：
- 结论必须与总起句形成"镜像"对应
- 使用确定性语言（"符合/不符合"、"成立/不成立"）
- 不得引入新的论证内容

**示例**：

```
因此，甲将乙推入河中的行为符合"杀害他人"这一构成要件行为要素。
```

### 2.3 涵摄四步法的形式化表达

```
┌─────────────────────────────────────────────┐
│ 总起句:  X的行为可能构成α中的要素E             │
├─────────────────────────────────────────────┤
│ 定义:    要素E是指满足条件D₁∧D₂∧...∧Dₙ        │
├─────────────────────────────────────────────┤
│ 归入:    事实F满足D₁（因为...）                │
│          事实F满足D₂（因为...）                │
│          ……                                   │
│          事实F满足Dₙ（因为...）                │
├─────────────────────────────────────────────┤
│ 结论:    X的行为(不)符合要素E                  │
└─────────────────────────────────────────────┘
```

## 三、构成要件符合性内部结构

### 3.1 客观构成要件

#### 3.1.1 行为主体

- **一般主体**：任何自然人
- **特殊主体**：身份犯中的特定身份要求
  - 真正身份犯：身份是犯罪成立的基础
  - 不真正身份犯：身份仅影响刑罚轻重

#### 3.1.2 构成要件行为

**作为犯**：

- 定义：以积极的身体动作实现构成要件的行为
- 检验重点：行为是否符合条文描述的行为类型

**不作为犯**：

- 真正不作为犯：法律明文规定不作为义务
- 不真正不作为犯：保证人地位 + 等价性

不真正不作为犯的特殊检验要素：

| 要素 | 内容 |
|------|------|
| 保证人地位 | 法律规定、合同约定、先行危险行为、紧密生活共同体、危险源监督 |
| 作为可能性 | 行为人在具体情境下有能力实施救助行为 |
| 等价性 | 不作为在规范评价上等同于作为 |

#### 3.1.3 行为对象

- 与保护法益的关系：行为对象是法益的载体
- 对象错误的处理：同类对象错误不阻却故意

#### 3.1.4 构成要件结果

- 结果犯：需要法益侵害结果的实际发生
- 行为犯：行为完成即既遂，无需特定结果
- 危险犯：
  - 具体危险犯：需证明具体危险状态的现实发生
  - 抽象危险犯：行为类型本身即被推定具有危险性

#### 3.1.5 因果关系与客观归责

**因果关系——条件说**：

```
检验公式：若无行为人的行为，结果是否仍然发生？
- 若结果不发生 → 行为是结果的原因（因果关系+）
- 若结果仍发生 → 行为不是结果的原因（因果关系-）
```

条件说的修正：
- 择一因果关系：假设去掉任一行为结果仍发生，但两个行为共同导致结果
- 累积因果关系：单独行为不足以导致结果，共同才能
- 超越因果关系：后行为切断前行为的因果流程

**客观归责**：

```
三步检验：
1. 行为是否制造了法所不容许的风险？
2. 该风险是否在结果中实现？
3. 结果是否在构成要件的效力范围内？
```

客观归责排除事由：

| 排除事由 | 说明 | 阶层 |
|----------|------|------|
| 风险降低 | 行为人的行为降低了既有风险 | 第1步 |
| 允许的风险 | 日常生活或合规活动中被社会普遍接受的风险 | 第1步 |
| 风险未实现 | 结果由其他独立风险实现 | 第2步 |
| 合法替代行为 | 即使合法行为也会导致相同结果 | 第2步 |
| 自我答责 | 被害人自我危殆 | 第3步 |
| 他人答责领域 | 第三人的专属职责范围 | 第3步 |

> **🔀 学派双轨提示（客观归责的本土化）**：立场 B（结果无价值论）主张应将客观归责理论的若干内容**分散安置于因果关系、实行行为、主观构成要件**之中，而**不是单独设立"客观归责"层**。具体而言：(1) 风险创设 → 实行行为的危险性；(2) 风险实现 → 因果关系的限缩；(3) 规范保护目的 / 自我答责 → 故意/过失的范围限定。本技能默认采**整合的客观归责论**（立场 A），但 agent 在用户配置切换至立场 B 或案件需要援引分散安置进路时，应说明立场切换并保持术语一致。

### 3.1.6 因果关系分析——介入因素强制检视

当因果链条中出现以下**任一**情形时，必须启动本检视（在条件公式和客观归责三步检验之外追加独立分析层）：

**触发条件**：

| 编号 | 介入类型 | 示例 |
|------|----------|------|
| I1 | 被害人实施了自主行为 | 自杀、自残、闯红灯、拒绝治疗、逃跑中遭遇事故 |
| I2 | 第三人实施了独立的介入行为 | 路人施救不当、第三人趁机加害、医疗过失 |
| I3 | 自然力/偶然事件介入 | 地震、洪水、意外爆炸 |

**检视步骤**：

```
(a) 识别介入因素的性质与类型（I1/I2/I3）
(b) 判断介入因素的"异常性"
    — 该介入是否超出了生活经验的通常预期？
    — 异常 → 倾向中断因果链
    — 不异常（属于先行行为的典型附随后果）→ 不中断
(c) 判断介入因素的"独立性"
    — 该介入是否由先行行为所创设的风险"典型地"引发？
    — 独立 → 倾向中断因果链
    — 非独立（是先行行为的自然延伸）→ 不中断
(d) 被害人介入行为的特殊检视（仅 I1 类型）：
    适用"被害人自我答责"原则的三个条件：
    (1) 被害人具有完全的自我决定能力
    (2) 被害人对危险具有充分认识
    (3) 被害人基于自由意志自主决定
    → 三项条件均满足：因果链条中断，结果不归责于行为人
    → 任一条件不满足：需进一步分析行为人对结果的归责程度
(e) 输出结论：
    → [归责成立] 介入因素不中断因果链
    → [因果链条中断，归责不成立] 介入因素异常且独立
    → [部分归责] 仅基本犯成立，不成立结果加重犯
```

**典型应用的案例模式**：

| 案例模式 | 介入因素分析 | 归责结论 |
|----------|-------------|----------|
| 强奸后被害人自杀 | 介入因素"异常"且"独立"（非强奸的典型附随后果） | 不归责为结果加重犯 |
| 嘲讽/侮辱后被害人跳楼 | 若嘲讽对跳楼有"支配性作用"→ 成立故意杀人；若仅为诱因 → 不归责 | 视具体支配程度而定 |
| 挑衅后被害人闯红灯被撞 | 被害人自我答责三条件均满足 | 因果链条中断 |
| 伤害后被害人拒绝治疗致死亡 | 需判断拒绝治疗是否"异常"——理性人是否会做同样选择 | 视具体情境而定 |

### 3.2 主观构成要件

#### 3.2.1 故意

**故意的二元结构**：

| 要素 | 内容 | 直接故意Ⅰ | 直接故意Ⅱ | 间接故意 |
|------|------|-----------|-----------|----------|
| 认识要素 | 对客观构成要件要素的认识 | 确定认识 | 确定认识 | 可能性认识 |
| 意志要素 | 对结果发生的意志态度 | 积极追求 | 确定接受 | 容忍/放任 |

**故意的认识对象**：

- 必须认识到所有客观构成要件要素
- 规范性构成要件要素：外行人平行评价（以外行人视角判断行为人是否认识到该要素的社会意义）
- 描述性构成要件要素：事实性认识即可

**故意的认识时点**：行为时

#### 3.2.2 主观超过要素

- 目的犯中的特定目的：如盗窃罪中的"非法占有目的"
- 表现犯中的特定内心状态（如伪证罪中"明知虚假"的内心状态，外部行为必须表达出该内心状态）
- 倾向犯中的特定倾向（如强制猥亵罪中的性倾向，行为须表现出特定的主观倾向）

#### 3.2.3 事实认识错误（构成要件错误）

> 事实认识错误若阻却故意，对应中国《刑法》第 14 条故意要件不满足，可能转入第 15 条过失或不构成犯罪。

| 错误类型 | 效果 | 说明 |
|----------|------|------|
| 对象错误 | 同类不阻却故意 | 同等价值对象间的错误 |
| 打击错误 | 争议：具体符合说vs法定符合说 | 行为偏差导致打击非预期对象 |
| 因果流程偏差 | 非重大偏差不阻却故意 | 实际因果流程与预想不一致 |
| 事实认识错误 | 阻却故意（学理通说，对应中国《刑法》第14条故意要件不满足） | 不知构成要件事实 |

> **🔀 学派双轨提示（打击错误的处理）**：本技能默认采**具体符合说**（立场 A）——A 想杀 X 误中 Y，对 X 成立故意杀人未遂、对 Y 成立过失致人死亡，依想象竞合处理。**立场 B 与传统通说采法定符合说**——A 直接成立故意杀人既遂（同一构成要件内的对象等价）。两说在量刑上区别不大但定罪结构不同；agent 写作时应明示所采立场（按 user-config 或主轨）并贯彻到底。

### 3.3 故意既遂归责对应检验

**核心原则**：行为人主观想象的因果流程与客观实际发生的因果流程之间，必须具有本质上的对应关系。

**检验步骤**：

```
1. 客观面：行为 → 因果关系 → 结果 → 客观归责 [均满足]
2. 主观面：故意覆盖了客观面的所有要素？
3. 对应检验：主观认识的因果流程与客观实际流程是否本质一致？
   - 一致 → 故意既遂(+)
   - 非本质偏差 → 故意既遂(+)
   - 本质偏差 → 故意既遂(-)，转入未遂+过失竞合检验
```

**关键判断标准**：因果流程偏差是否属于"一般生活经验范围内可预见的偏差"。

## 四、违法性内部结构

### 4.1 基本逻辑：推定与推翻

```
构成要件符合性(+) 
    → 违法性推定成立
    → 检验是否存在正当化事由
        → 无正当化事由 → 违法性确认(+)
        → 有正当化事由 → 违法性排除(-)
```

### 4.2 正当化事由总览

#### 4.2.1 正当防卫（刑法第20条）

**构成要件**：

| 要素 | 内容 |
|------|------|
| 防卫情状 | 存在正在进行的不法侵害 |
| 防卫行为 | 针对不法侵害人本人实施 |
| 防卫必要性 | 在有效制止侵害的手段中选择最轻微的 |
| 防卫适当性 | 防卫强度与侵害强度之间不存在明显失衡 |

**防卫限度的社会伦理限制**：
- 轻微侵害：仅允许轻微防卫
- 可归责于防卫者的侵害前因
- 来自无责任能力者的侵害：退避义务优先
- 保证人关系中的特殊限制

> **🔀 学派双轨提示（§20 第 3 款"特殊正当防卫"的性质）**：中国《刑法》第 20 条第 3 款规定"对正在进行行凶、杀人、抢劫、强奸、绑架以及其他严重危及人身安全的暴力犯罪，采取防卫行为，造成不法侵害人伤亡的，不属于防卫过当，不负刑事责任"。
>
> - **本技能主轨（"特殊正当防卫说"——立场 A 通说）**：将其作为正当防卫一般规定（第 1 款 + 第 2 款）之外的**特殊正当防卫**——对特定严重暴力犯罪不存在防卫过当问题。
> - **立场 B（"注意规定说"）**：第 3 款是**注意规定**而非创设性规定——"严重危及人身安全的暴力犯罪"本身就是无限威胁，按第 1-2 款本来也不会成立防卫过当；第 3 款只是把这一结论显性化，避免实务误判。
>
> 影响：采"特殊正当防卫说"时，第 3 款列举的"行凶、杀人、抢劫、强奸、绑架"必须达到法定列举的具体罪名才适用；采"注意规定说"时，关键不在于罪名标签而在于**侵害的严重性是否达到无限威胁**。本技能默认主轨，agent 在涉及第 3 款适用且 user-config 未切换至立场 B 时，仍应显性讨论"注意规定说"的影响。

#### 4.2.2 紧急避险（刑法第21条）

| 要素 | 内容 |
|------|------|
| 紧急状态 | 法益面临现实的、紧迫的危险 |
| 避险行为 | 损害第三人合法利益 |
| 利益衡量 | 所保全利益明显大于所损害利益 |
| 适当性 | 避险手段是唯一可行且损害最小的 |
| 避险意思 | 主观上认识到紧急状态并为避险而行为 |

#### 4.2.3 被害人承诺

**有效要件**：
- 法益可处分性（个人法益且非生命法益）
- 承诺能力（理解能力和判断能力）
- 承诺的自愿性（无强制、欺诈、重大错误）
- 承诺的时点（行为时或行为前）
- 承诺的外部表示
- 行为人认识到承诺的存在

#### 4.2.4 推定的承诺

适用条件：无法及时获得明示承诺，且根据客观合理判断，法益主体如知情将会同意。

#### 4.2.5 法令行为与其他正当化事由

- 法令行为（依据法律授权或命令实施的行为，如强制执行、依法拘留）
- 正当业务行为（在合法业务范围内依行业规范实施的行为，如医生的合规医疗行为）
- 自救行为（权利人在无法及时获得公力救济时，以适当方式自行恢复权利的行为）
- 义务冲突（两项法律义务同时存在且无法同时履行，履行其中一项而违反另一项的情形）

### 4.3 正当化事由的主观要素

> **行为无价值二元论的核心要求**：正当化不仅要求客观上存在正当化情状，还要求行为人主观上认识到该情状并基于该认识而行为。

```
正当化事由成立 = 客观正当化情状(+) ∧ 主观正当化要素(+)
```

**缺乏主观正当化要素的法律效果**：

| 立场 | 处理方式 | 说明 |
|------|----------|------|
| 通说 | 未遂犯 | 客观上无法益侵害，但主观上有不法意志 |
| 少数说 | 既遂犯（但酌情从轻） | 主观恶性完整 |
| 本技能采纳 | 未遂犯说 | 结果无价值缺失，仅有行为无价值 |

> **🔀 学派双轨提示（偶然防卫——结果无价值论的"客观说"）**：偶然防卫指**客观上存在正当化情状但行为人主观无认识**（如 A 出于报复枪杀 B，恰巧 B 正在杀 C 的过程中，客观上保护了 C 的生命）。
>
> - **本技能主轨（立场 A 通说"未遂犯说"）**：成立未遂犯——客观上虽无可罚的法益侵害，但主观上的不法意志独立于结果而存在。
> - **立场 B（"客观正当化要素说"/"无罪说"）**：客观上既然存在正当化情状即阻却违法，行为人主观无认识不影响违法阻却；A 的行为根本不成立犯罪。
>
> 这是结果无价值论与行为无价值二元论最尖锐的对立点。agent 在涉及偶然防卫案型时，**必须**显性展开两说并按 user-config 选边。本技能默认主轨"未遂犯说"，理由是：单纯结果决定论会使刑法的行为规范功能丧失，且与中国《刑法》第 23 条未遂犯一般性规定相协调。

## 五、有责性内部结构

### 5.1 基本定位

有责性是对行为人的**个人可非难性**的判断：行为人在具有选择合法行为的可能性时，仍然选择了不法行为。

### 5.2 责任能力

#### 5.2.1 年龄标准

| 年龄 | 责任能力状态 | 法律依据 |
|------|-------------|----------|
| 不满14周岁 | 完全无责任能力 | 刑法第17条第1款 |
| 已满14不满16周岁 | 相对有责任能力（仅对8类重罪负责） | 刑法第17条第2款 |
| 已满16不满18周岁 | 完全有责任能力（从轻/减轻） | 刑法第17条第3款 |
| 已满18周岁 | 完全有责任能力 | — |

#### 5.2.2 精神状态标准

**生物学-心理学混合标准**：

```
生物学要素（诊断）：
- 精神病
- 精神发育迟滞
- 其他精神障碍

  ＋

心理学要素（功能影响）：
- 不能辨认行为的违法性（认识能力丧失）
- 不能控制自己的行为（控制能力丧失）
```

#### 5.2.3 原因自由行为

当行为人故意或过失使自己陷入无责任能力状态，再于该状态下实施犯罪行为：

- 原因行为时具有完全责任能力
- 原因行为时已有犯罪故意或过失
- 原因行为与结果行为之间具有关联性

### 5.3 违法性认识可能性

> 学理多数说采责任说处理。

**禁止错误的处理**：

| 类型 | 效果 |
|------|------|
| 不可避免的禁止错误 | 排除责任（学理通说采责任说；中国《刑法》对违法性认识无明文，第16条是不可抗力与此无关） |
| 可避免的禁止错误 | 不排除责任，可酌情从轻 |

**可避免性判断标准**：
- 行为人是否有理由怀疑自己行为的合法性
- 是否有查询法律的合理期待
- 行为领域的专业性程度

### 5.4 期待可能性

当具体情境下不能期待行为人做出合法行为选择时，排除有责性。

**适用场景**：
- 超法规免责的紧急避险
- 不可抗力情境下的强制
- 良心冲突的极端情形（高度争议）

### 5.5 容许构成要件错误

> 中国法上"假想防卫"通常作为认识错误问题处理，按学理多数说排除故意（构成过失犯）。

**问题情境**：行为人误以为存在正当化事由的前提事实（如假想防卫）。

**学说争议**：

| 学说 | 处理方式 | 体系位置 |
|------|----------|----------|
| 严格责任说 | 不影响故意，仅影响有责性 | 有责性 |
| 限制责任说（通说） | 类推适用构成要件错误规则，排除故意 | 构成要件/有责性（争议） |
| 消极构成要件要素说 | 直接排除构成要件故意 | 构成要件 |

**本技能采纳**：限制责任说——将容许构成要件错误置于有责性阶层中的"责任故意"层面处理，排除故意责任，但可能成立过失犯。

> **🔀 学派双轨提示（容许构成要件错误体系位置）**：立场 A 与立场 B 在**结论**上高度一致（都认为应阻却故意、按过失犯处理），但**体系位置**有分歧：
>
> - **立场 A 主轨（本技能 default）**：限制责任说——错误置于**有责性**层面（"责任故意"），与构成要件故意分离；
> - **立场 B 变体**：限制责任说的另一变体——直接认为**构成要件故意**被阻却，更接近"消极构成要件要素说"；
> - **严格责任说**（少数说）：错误不影响故意，仅作为责任减免事由。
>
> 在"假想防卫致人死亡"等典型案例中，立场 A、立场 B 结论一致；体系位置差异主要影响共犯责任的连带性（限制责任说下，正犯虽阻却故意，共犯仍可成立）。agent 写作时按 user-config 选定立场或本技能主轨展开即可。

## 六、其他犯罪要件

### 6.1 客观处罚条件

- 不属于不法与责任的范畴
- 不需要被故意所覆盖
- 存在争议：是否影响共犯的可罚性

**示例**：
- 丢失枪支不报罪中的"造成严重后果"
- 某些数额犯中的"数额较大"（有争议）

### 6.2 未遂与不能犯

> **🔀 学派双轨提示（不能犯/未遂区分）**："不能犯"指行为从一开始就不可能造成既遂结果的情形（如用空枪杀人、盗窃空保险箱）。两派的核心分歧在于判断"危险性"的标准：
>
> - **立场 A（本技能 default）**：具体危险说——以行为时**一般人能感知的事实 + 行为人的特别认知**为基础，判断是否存在具体危险。有具体危险→未遂犯；无具体危险→不能犯（不罚）。
> - **立场 B**：客观危险说——以行为时**全部客观事实**（含事后查明的事实）为基础判断。客观上完全无危险→不能犯（不罚）。
>
> 立场 A 下未遂范围更宽，立场 B 下不能犯范围更宽。agent 写作时按本技能主轨（立场 A）展开即可。

### 6.3 刑罚解除事由

#### 6.3.1 犯罪中止

```
检验位置：三阶层之后的独立检验层面
（注意：不属于构成要件、违法性或有责性中的任何一个！）

检验步骤：
1. 是否属于未遂阶段？（已着手但未既遂）
2. 中止的自愿性——弗兰克公式（能达目的而不欲 = 自愿；欲达目的而不能 = 非自愿）
3. 中止的有效性：
   - 未了未遂：放弃继续实施即可
   - 已了未遂：须积极阻止结果发生
4. 中止成功 → 免除处罚
```

#### 6.3.2 其他刑罚解除事由

- 亲属间盗窃的免诉事由
- 自首、立功等法定量刑情节（注意：部分为量刑而非免刑）

## 七、涵摄链与嵌套涵摄

### 7.1 涵摄链

一个完整罪名的检验是由多个涵摄单元组成的"链"：

```
罪名检验 = 涵摄₁(主体) + 涵摄₂(行为) + 涵摄₃(对象) + 涵摄₄(结果) 
          + 涵摄₅(因果关系) + 涵摄₆(客观归责) + 涵摄₇(故意) + ...
```

每一个涵摄单元都遵循"总起句-定义-归入-结论"四步法。

### 7.2 嵌套涵摄

**触发条件**：在进行某一要素的归入时，发现需要先解决一个子问题。

**操作方式**：

```
┌── 外层涵摄：要素A的检验 ──────────────────────────┐
│  总起句: 甲的行为可能符合要素A                       │
│  定义:   要素A是指满足条件D                          │
│  归入:                                              │
│    ┌── 内层涵摄：条件D中的子问题D₁ ───────────┐    │
│    │  总起句: 此处需确认是否满足D₁              │    │
│    │  定义:   D₁是指...                         │    │
│    │  归入:   案件事实中...                      │    │
│    │  结论:   D₁(+)/(-)                         │    │
│    └──────────────────────────────────────────┘    │
│  （回到外层）继续检验其余条件...                       │
│  结论: 要素A(+)/(-)                                   │
└──────────────────────────────────────────────────────┘
```

### 7.3 实践指引

1. **嵌套深度限制**：原则上不超过三层嵌套，否则应重新组织论证结构
2. **嵌套标记**：使用缩进和标题层级明确标记嵌套的开始与结束
3. **嵌套触发时机**：
   - 定义中包含另一个需要解释的法律概念
   - 归入中遇到事实是否符合定义存在疑问
   - 需要引入学说争议来解决归入问题
4. **避免过度嵌套**：如果子问题可以独立成段，优先独立处理后引用结论

## 八、小结：内在结构的全景图

```
犯罪成立 = 
  构成要件符合性(+)
    ├─ 客观构成要件
    │   ├─ 行为主体
    │   ├─ 构成要件行为（作为/不作为）
    │   ├─ 行为对象
    │   ├─ 构成要件结果
    │   ├─ 因果关系（条件说）
    │   ├─ 客观归责（三步检验）
    │   └─ 介入因素强制检视（§3.1.6）
    └─ 主观构成要件
        ├─ 故意（认识+意志）
        ├─ 主观超过要素
        └─ 归责对应检验
  ＋
  违法性(+)
    └─ 无正当化事由
        ├─ 正当防卫(-)
        ├─ 紧急避险(-)
        ├─ 被害人承诺(-)
        └─ 其他正当化事由(-)
  ＋
  有责性(+)
    ├─ 责任能力(+)
    ├─ 违法性认识可能性(+)
    ├─ 期待可能性(+)
    └─ 无容许构成要件错误
  ＋
  [其他要件]
    ├─ 客观处罚条件(如有)
    └─ 无刑罚解除事由

全部满足 → 犯罪成立 → 进入量刑与竞合处理
```
