# 论证深度比例原则

> **核心命题**：论证的深度应当与问题的争议程度成正比——明显成立/不成立者一句带过，存在合理争议者完整涵摄，核心争议要件穷尽方法论工具。
>
> **隐喻**：优秀的鉴定式报告如同五花肉——肥（详细论证）瘦（简洁陈述）相间、节奏分明。全肥则冗长无重点；全瘦则单薄无深度；肥瘦倒置（显然者详写、争议者略写）则本末倒置。

---

## 一、三层分类体系（核心）

| 层级 | 要素特征 | 论证模式 | 典型篇幅 |
|------|----------|----------|----------|
| **第一层** | 明显成立 / 不成立 | 裁判式 | 1–2 句 |
| **第二层** | 存在合理争议 | 标准鉴定式 · 涵摄四步法 | 1–2 段（150–400 字） |
| **第三层** | 核心争议焦点 | 深度鉴定式 = 涵摄 + 法律解释 + 学说争议 + 反对说回应 | 多段 / 整节 |

---

## 二、层级判定决策（四问）

对每个要件依次回答四问，依综合结果定层级：

| 诊断问题 | 提问 | 结果指向 |
|---|---|---|
| Q1 核心语义检验 | 案件事实是否直接、无疑义地落入概念核心区？ | 是 → 倾向第一层 |
| Q2 概念可把握性 | 概念边界是否依文义即可清晰把握，无需解释方法介入？ | 是 → 倾向第一层；否 → 倾向第二/三层 |
| Q3 学说/实务分歧 | 学界或司法实务对此情形是否存在分歧？ | 重大分歧 → 第三层 |
| Q4 决定性检验 | 该要件的判断是否决定罪名成立或重要走向？ | 是 + Q3 有分歧 → 第三层 |

**综合判断矩阵**：

| Q1 | Q2 | Q3 | Q4 | 层级 |
|---|---|---|---|---|
| 是 | 是 | — | — | 第一层 |
| 否 | — | 无 | 否 | 第二层 |
| 否 | — | 有 | 是 | 第三层 |
| — | — | 重大分歧 | 是 | 第三层 |

## 三、各层论证格式

### 3.1 第一层（裁判式）

> 甲系年满 25 周岁、精神正常的自然人，具有完全刑事责任能力。

> 乙（有生命的自然人）系适格行为对象。

**典型属于第一层**：行为主体（成年精神正常自然人）、行为对象（明确）、结果发生（题目明示）、责任能力（无疑义）、违法性（无任何正当化情节线索）。

**注意**：第一层要素不得跳过，必须有一句话确认。题目暗含异常（如行为人年龄接近 14 周岁）时不得归入第一层。

### 3.2 第二层（标准鉴定式 · 涵摄四步法）

```
[总起句]   甲的 X 行为可能符合"…"的构成要件要素。
[定义]     所谓"…"，是指…（依文义/体系/目的展开）。
[归入]     此处，案件事实显示…，符合/不符合上述定义之要素。
[结论]     因此，甲的行为（不）符合"…"要素。
```

**典型属于第二层**：构成要件行为是否"足以致命"、因果关系是否被介入因素中断、客观归责中风险是否实现、间接故意 vs 有认识过失、未遂的"着手"判定、共同正犯的"共同行为决意"。

### 3.3 第三层（深度鉴定式）

```
[标准涵摄四步法]
   +
[法律解释展开（文义 / 体系 / 历史 / 目的；遇分则解释调用 interpretation-toolkit.md 下编）]
   +
[学说争议展开：甲说 / 乙说 / 丙说 → 对各立场逐一推演本案 → 选择立场 + 积极理由]
   +
[反对说回应（实质性回应，非简单重复己方观点）]
   +
[选定立场的贯彻适用]
```

**典型属于第三层**：偶然防卫、容许构成要件错误、§20 第 3 款的扩张/限缩、客观归责的具体应用、抢劫罪的复行为构造、注意规定 vs 法律拟制鉴别、兜底条款的限缩适用。

## 四、学说争议的展开规则

### 4.1 "结论一致则不问理由"

> **当所有有代表性的学说立场对同一问题均得出相同结论时，无需展开学说争议；但应在脚注点明争议存在。**

**典型用法**：

> 此处虽存在具体符合说与法定符合说之争（关于打击错误），但无论采何说，本案均得出甲对丙具有故意的结论 [^注]，故不展开。
>
> [^注]：具体符合说下：…（故意+）；法定符合说下：…（故意+）。

**例外**：若后续问题的处理取决于此处采用哪一说，则不得跳过，必须显性选边。

### 4.2 结论不一致 · 必须展开

| 争议类型 | 展开要求 |
|---|---|
| 直接影响定罪（罪 / 非罪、A 罪 / B 罪） | 全面展开 + 明确选择 + 贯彻适用 |
| 影响犯罪形态（既遂 / 未遂、正犯 / 共犯） | 标准展开 + 选择 + 适用 |
| 影响量刑 | 适度展开 + 选择 |
| 仅影响理论解释、不影响结论 | 简要提及 + 不展开 |

### 4.3 选择立场的论证义务

选择某一学说时，须提供**积极理由**（不得仅否定其他立场）：体系融贯性 / 法益保护充分性 / 罪刑法定遵守 / 实践可操作性。

### 4.4 反对说展开深度分级

| 反对说影响 | 展开要求 |
|---|---|
| 低（不影响结论） | 一句带过或脚注简述 |
| 中（影响形态/量刑） | 1 段实质回应 |
| 高（影响罪/非罪） | 充分展开 + 立场对比 + 实质回应 |
| 司法判例支持的反对说 | 必须事实区分或正面回应 |

### 4.5 核心争议标准呈现模板

对于直接影响定罪的核心争议（§4.2 第一行），推荐使用以下结构化呈现模板：

```markdown
| 学说 | 核心主张 | 主要论据 |
|------|----------|----------|
| 甲说（通说） | [一句话概括] | [关键论据] |
| 乙说（少数说） | [一句话概括] | [关键论据] |
| 丙说（极少数说） | [一句话概括] | [关键论据] |

**分析与选择**：
[从体系融贯性、法益保护、罪刑法定、实践可操作性等维度比较]

**本文立场**：采纳[X]说。理由如下：
[积极理由，不得仅否定其他立场]

**反对说回应**：
[针对最有说服力的反对说进行实质回应]

**立场贯彻**：
[将所选立场贯彻到后续涵摄中，保持逻辑一致]
```

agent 在处理核心争议时**应主动使用此模板**，确保学说分歧的呈现结构统一、论证完整。

## 五、自查清单

```
□ 是否对每个要件都做了层级判断？
□ 有无将争议要素错误归入第一层（漏写）？
□ 有无将显然要素错误归入第三层（堆料）？
□ 第二层要素是否均完成涵摄四步法？
□ 第三层要素是否包含法律解释 + 学说展开 + 反对说回应？
□ 学说争议结论一致时是否在脚注点明？结论不一致时是否充分展开？
□ 立场选择是否给出积极理由？整篇是否贯彻同一立场（或显性切换并说明）？
□ 详略节奏：核心争议是否占全文主要篇幅？有无前重后轻？
```

## 六、篇幅参考

|---|---|
| 事实单元划分 + 参与人列举 | 5–10% |
| 核心罪名检验 | 50–65% |
| ┗ 第三层要素 | 占核心检验的 50–70% |
| ┗ 第二层要素 | 占核心检验的 25–40% |
| ┗ 第一层要素 | 占核心检验的 5–15% |

| 竞合处理 | 5–10% |
| 全案结论 | 3–5% |

**总字数指引**：默认 ≤ 30,000 字；案情简单时压缩至 8,000–15,000 字。

## 七、常见失衡与修正

| 失衡 | 表现 | 修正方向 |
|---|---|---|
| 详略倒置 | 显然要件长篇、核心争议草草 | 重新分配篇幅 |
| 平均主义 | 所有要件等量齐观 | 按四问重判层级 |
| 头重脚轻 | 前半极详、后半潦草 | 控制前半节奏 |
| 偏废违法 / 有责 | 构成要件详、违法性/有责性一笔带过 | 如有正当化/免责争议须充分展开 |
| 学说堆砌 | 对不影响结论的争议也全面展开 | 适用"结论一致不问理由" |
