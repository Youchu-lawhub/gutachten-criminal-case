# 外在结构：五步分析法与罪名检索顺序

---

## 一、五步分析法

### 1.1 总览

五步分析法是处理复杂刑法案例的**宏观框架**，规定了从拿到案件事实到得出最终结论的完整路径：

```
┌─────────────────────────────────────────────────────────────┐
│  第一步：事实单元划分                                         │
│         ↓                                                    │
│  第二步：参与人员列举                                         │
│         ↓                                                    │
│  第三步：涉嫌犯罪检验                                         │
│         ↓                                                    │
│  第四步：犯罪竞合处理                                         │
│         ↓                                                    │
│  第五步：全案结论                                             │
└─────────────────────────────────────────────────────────────┘
```

---

### 1.2 第一步：事实单元划分

#### 1.2.1 划分目的

将复杂案件事实分解为相对独立的分析单元，使检验具有清晰的组织结构，避免遗漏。

#### 1.2.2 划分标准

**核心标准——行为标准**：

以一个独立的行为（或一组密切关联的行为）所产生的一个独立结果为一个事实单元。

**操作规则**：

| 规则编号 | 规则内容 | 说明 |
|----------|----------|------|
| R1 | 一行为一结果 = 一单元 | 最基本的划分标准 |
| R2 | 预备行为不独立成单元 | 将预备行为与实行行为归入同一单元 |
| R3 | 共同犯罪归入同一单元 | 正犯与共犯在同一单元中检验 |
| R4 | 时间断裂可分割 | 明显不同时间段的不同行为应分属不同单元 |
| R5 | 行为转换即新单元 | 行为方式/目标发生根本变化时开启新单元 |
| R6 | 连续行为不拆分 | 基于同一犯罪决意的连续行为归入同一单元 |

#### 1.2.3 划分步骤

```
步骤1：通读全部案件事实，标记所有行为节点
步骤2：识别每个行为节点对应的结果（或未遂状态）
步骤3：按时间顺序排列行为-结果配对
步骤4：应用上述规则R1-R6，合并或分割
步骤5：为每个事实单元赋予中性描述性标签
```

#### 1.2.4 中性标签命名规范

- **使用客观描述**，不预设法律评价
- **避免使用罪名**作为标签

| 正确示例 | 错误示例 | 原因 |
|----------|----------|------|
| "甲对乙的殴打行为" | "甲的故意伤害行为" | 预设了罪名结论 |
| "丙取走丁的手机" | "丙盗窃丁的手机" | 预设了犯罪性质 |
| "戊驾车撞击行人" | "戊的交通肇事" | 预设了过失 |
| "庚向辛提供刀具" | "庚帮助杀人" | 预设了共犯形态 |

#### 1.2.5 特殊情形处理

**情形一：一行为多结果**

```
甲一拳打到乙，同时将乙手中的花瓶打碎
→ 归入同一事实单元（一个自然行为）
→ 在竞合阶段处理（想象竞合）
```

**情形二：多行为一结果**

```
甲先下毒（第一次投毒不致死），后又投毒（累计致死）
→ 判断是否基于同一犯罪决意
→ 是：归入同一事实单元
→ 否：分属不同单元
```

**情形三：中间介入其他行为人**

```
甲将乙打伤后离开，丙路过对乙实施抢劫
→ 甲的伤害行为 = 单元一
→ 丙的抢劫行为 = 单元二
→ 不同行为人、不同犯罪决意、不同时间
```

### 1.3 第二步：参与人员列举

#### 1.3.1 列举原则

**全面列举**：在每个事实单元中，列出所有参与行为的自然人。

**包含范围**：
- 直接实施者
- 间接利用者
- 教唆者
- 帮助者
- 甚至无刑事责任能力者（如被利用的未成年人）

#### 1.3.2 分析顺序——密接者优先原则

```
基本原则：越"接近"法益侵害结果的人越优先分析

具体顺序：
1. 直接正犯 —— 直接实施构成要件行为者
2. 间接正犯 —— 利用他人为工具者
3. 共同正犯 —— 共同实施者
4. 教唆犯 —— 引起犯意者
5. 帮助犯 —— 提供便利者
```

#### 1.3.3 间接正犯中的特殊顺序

在间接正犯情形中：

```
首先分析直接行为者（疑似被利用者）
→ 确认其是否具有正犯资格/是否缺乏故意/是否缺乏责任能力
→ 然后分析幕后操控者
→ 判断其是否构成间接正犯
```

**原因**：只有先确认直接行为者"不构成正犯"（或存在归责障碍），才能论证幕后者的间接正犯地位。

#### 1.3.4 无刑事责任能力者的分析

即使行为人因年龄（不满14周岁）或精神状态而不具有刑事责任能力，仍应完成构成要件与违法性的检验：

- 这关系到对其行为的正当防卫是否成立的判断
- 这关系到幕后利用者是否构成间接正犯的判断
- 在有责性阶段否定即可

### 1.4 第三步：涉嫌犯罪检验

#### 1.4.1 核心地位

这是五步分析法的**核心步骤**，占据报告篇幅的绝大部分。对每一参与人在每一事实单元中，检验所有可能涉及的罪名。

#### 1.4.2 检验框架

对每一个罪名的检验，严格遵循三阶层体系：

```
犯罪检验：
├─ I. 构成要件符合性
│   ├─ 1. 客观构成要件
│   │   ├─ 行为主体
│   │   ├─ 构成要件行为
│   │   ├─ 行为对象
│   │   ├─ 结果
│   │   ├─ 因果关系
│   │   └─ 客观归责
│   └─ 2. 主观构成要件
│       ├─ 故意/过失
│       └─ 主观超过要素
├─ II. 违法性
│   └─ 正当化事由检验
├─ III. 有责性
│   ├─ 责任能力
│   ├─ 违法性认识可能性
│   └─ 期待可能性
└─ [IV. 其他：客观处罚条件/刑罚解除事由]
```

#### 1.4.3 犯罪形态的检验变体

**既遂犯**：标准检验流程

**未遂犯**：

```
I. 构成要件符合性
   1. 犯罪决意
      - 完成罪的全部主观要素
      - 对客观构成要件要素的认识与意志
   2. 着手实行
      - 标准：行为人主观上认为已开始实现构成要件
      - 客观上已直接危及法益
   3. 未达既遂
      - 结果未发生/构成要件未完全实现
II. 违法性
III. 有责性
[IV. 中止犯检验——刑罚解除事由]
```

**共犯**：

```
I. 构成要件符合性
   1. 正犯的不法行为
      - 正犯至少达到构成要件符合+违法（学理通说采限制从属性说）
   2. 共犯行为
      - 教唆犯：引起正犯犯意
      - 帮助犯：为正犯提供物质或精神帮助
   3. 共犯故意
      - 双重故意：对自己的共犯行为+对正犯的不法行为
II. 违法性
III. 有责性
```

**不作为犯**：

```
I. 构成要件符合性
   1. 客观构成要件
      ├─ 构成要件结果的发生
      ├─ 不作为
      ├─ 作为可能性
      ├─ 保证人地位[不真正不作为犯]
      ├─ 因果关系（准因果关系：若行为则结果极有可能不发生）
      ├─ 客观归责
      └─ 等价性[不真正不作为犯]
   2. 主观构成要件
II. 违法性
III. 有责性
```

**过失犯**：

```
I. 构成要件符合性
   1. 客观构成要件
      ├─ 客观注意义务违反
      ├─ 客观可预见性
      ├─ 结果发生
      ├─ 因果关系
      ├─ 客观归责（注意义务违反关联性）
      └─ 规范保护目的
   2. 主观构成要件
      ├─ 主观可预见性
      └─ 主观可避免性
II. 违法性
III. 有责性
```

### 1.5 第四步：犯罪竞合处理

#### 1.5.1 竞合的体系位置

竞合处理在**同一事实单元内**完成，在所有罪名检验结束之后进行。

#### 1.5.2 竞合类型

| 竞合类型 | 判断标准 | 法律效果 |
|----------|----------|----------|
| 想象竞合 | 一行为触犯数罪名 | 从一重处断 |
| 法条竞合 | 法条间有逻辑包含关系 | 适用一法条 |
| 实质竞合 | 数行为触犯数罪名 | 数罪并罚 |
| 牵连犯 | 手段与目的/原因与结果 | 从一重处断（中国特色） |
| 吸收犯 | 一罪为另一罪的必经阶段 | 重行为吸收轻行为 |

#### 1.5.3 法条竞合的具体类型

```
法条竞合：
├─ 特别关系：特别法优于一般法
├─ 补充关系：主法条优于补充法条
├─ 吸收关系：全面评价法条优于被吸收法条
└─ 择一关系：相互排斥的法条
```

#### 1.5.4 竞合判断的操作流程

```
步骤1：确认该事实单元内有多少个罪名成立
步骤2：判断"行为个数"——自然行为单数还是复数？
步骤3：
  - 一行为+多罪名 → 想象竞合？法条竞合？
  - 多行为+多罪名 → 实质竞合？牵连犯？
步骤4：确定最终适用的法条与刑罚计算方式
```

### 1.6 第五步：全案结论

#### 1.6.1 格式要求

全案结论必须**逐人逐罪**列明每位参与人的最终刑事责任：

```
## 全案结论

### 一、甲的刑事责任
1. 甲以直接正犯方式构成刑法第234条第2款故意伤害罪（致人重伤），
   与刑法第263条抢劫罪（基本犯）想象竞合，从一重处断。
2. 甲另构成刑法第XXX条……，与前述实质竞合，数罪并罚。

### 二、乙的刑事责任
1. 乙以帮助犯方式构成刑法第234条第2款故意伤害罪（致人重伤）。
```

#### 1.6.2 结论要素

每一条结论必须包含以下要素：

| 要素 | 示例 |
|------|------|
| 参与形态 | 直接正犯/间接正犯/共同正犯/教唆犯/帮助犯 |
| 犯罪形态 | 既遂/未遂/中止/预备 |
| 法条编号 | 刑法第XXX条第X款第X项 |
| 罪名全称 | 故意杀人罪/盗窃罪等 |
| 加重/减轻情节 | 致人死亡/数额巨大/自首/立功等 |
| 竞合关系 | 与XX罪想象竞合/实质竞合 |

## 二、罪名检索顺序

### 2.1 基本排序原则

#### 2.1.1 四大排序规则

| 优先级 | 规则 |
|--------|------|
| 1 | 先正犯后共犯 |
| 2 | 先既遂后未遂 |
| 3 | 先故意后过失 |
| 4 | 先重罪后轻罪 |

#### 2.1.2 规则间的协调

当规则发生冲突时的处理：

```
规则1 > 规则2 > 规则3 > 规则4

即：正犯的未遂 优先于 共犯的既遂
    正犯的过失 优先于 共犯的故意（但实践中罕见）
```

### 2.2 法益导向检索法

#### 2.2.1 操作步骤

```
步骤1：识别案件事实中所有受到侵害或威胁的法益
        ├─ 个人法益：生命、身体、自由、名誉、财产、性自主权
        └─ 超个人法益：公共安全、国家机能、司法秩序、经济秩序

步骤2：按法益类型对应刑法分则章节
        ├─ 生命法益 → 第四章第一节（侵犯公民人身权利）
        ├─ 身体法益 → 第四章第一节
        ├─ 财产法益 → 第五章（侵犯财产罪）
        ├─ 自由法益 → 第四章第一节
        ├─ 公共安全 → 第二章
        └─ ……

步骤3：在对应章节中检索所有可能适用的罪名
        ├─ 基本构成
        ├─ 加重构成
        ├─ 减轻构成
        └─ 法律拟制条款

步骤4：排列检索到的罪名，确定检验顺序
```

#### 2.2.2 法益识别清单

| 法益类型 | 常见侵害方式 | 对应核心罪名 |
|----------|-------------|-------------|
| 生命 | 杀害、致死 | 故意杀人罪、过失致人死亡罪 |
| 身体完整性 | 伤害、虐待 | 故意伤害罪、虐待罪 |
| 人身自由 | 拘禁、绑架 | 非法拘禁罪、绑架罪 |
| 性自主权 | 强制性行为 | 强奸罪、强制猥亵罪 |
| 财产-占有 | 窃取、抢夺 | 盗窃罪、抢夺罪、抢劫罪 |
| 财产-整体 | 骗取、侵占 | 诈骗罪、侵占罪 |
| 公共安全 | 放火、爆炸 | 放火罪、爆炸罪 |
| 司法秩序 | 伪证、窝藏 | 伪证罪、窝藏罪 |
| 国家机能 | 贿赂、渎职 | 受贿罪、滥用职权罪 |

### 2.3 特别法与一般法的顺序

#### 2.3.1 基本规则

当多个罪名之间存在**特别法-一般法关系**时：

```
检验顺序：先检验特别法（特殊罪名），后检验一般法（普通罪名）
```

**原因**：如果特别法成立，一般法被排除（法条竞合中的特别关系）；如果特别法不成立，需要退回检验一般法。

#### 2.3.2 常见特别法-一般法关系

| 特别法（优先检验） | 一般法（后检验） | 关系说明 |
|-------------------|-----------------|----------|
| 抢劫罪 | 盗窃罪+故意伤害罪 | 复合行为犯 |
| 贪污罪 | 盗窃罪/诈骗罪/侵占罪 | 主体特殊 |
| 故意杀人罪 | 故意伤害罪（致死） | 故意内容特殊 |
| 保险诈骗罪 | 诈骗罪 | 对象特殊 |
| 交通肇事罪 | 过失致人死亡罪 | 场景特殊 |

### 2.4 无特别关系时的排序

当罪名之间**不存在**特别法-一般法关系时：

```
规则1：先检验法定刑更重的罪名
规则2：法定刑相当时，按行为的时间顺序
规则3：侵犯同一法益的罪名相邻排列
```

### 2.5 法律拟制条款的检索

#### 2.5.1 什么是法律拟制

法律拟制是指法律将本不符合某一罪名构成要件的行为，视同符合该罪名处理。

#### 2.5.2 常见法律拟制条款

| 条文 | 内容 | 效果 |
|------|------|------|
| 刑法第269条 | 犯盗窃等罪，为抗拒抓捕等当场使用暴力 | 以抢劫论 |
| 刑法第267条第2款 | 携带凶器抢夺 | 以抢劫论 |
| 刑法第196条第3款 | 盗窃信用卡并使用 | 以盗窃论 |
| 刑法第289条 | 聚众"打砸抢"致人伤残、死亡 | 以故意伤害/杀人论 |

#### 2.5.3 检索策略

在完成核心罪名检验后，必须额外检索：
- 该罪名条文中是否有"以XX罪论处"的转化条款
- 相关罪名是否有针对本案行为模式的拟制规定
- 司法解释中是否有"按照XX罪定罪处罚"的规定

### 2.6 犯罪形态的全面检索

对每一个可能涉及的罪名，必须考虑所有犯罪形态：

```
┌─ 既遂——标准检验
├─ 未遂——构成要件未完全实现时
├─ 预备——未达着手时（仅重罪）
└─ 中止——自愿放弃/阻止结果时
```

**检索规则**：

- 首先尝试既遂的检验
- 如果既遂不成立（如结果未发生或归责失败），转入未遂检验
- 未遂成立后，检验是否存在中止（刑罚解除事由）
- 预备犯仅在法律有明文规定时检验

### 2.7 参与形态的全面检索

对每一个参与人，必须考虑其可能的参与形态：

```
┌─ 直接正犯——亲自实施
├─ 间接正犯——利用他人
├─ 共同正犯——共同实施
├─ 教唆犯——引起犯意
└─ 帮助犯——提供帮助
```

## 三、参与人分析顺序

### 3.1 先正犯后共犯

**理论依据**：共犯从属性原则——共犯的成立以正犯的构成要件符合性+违法性为前提（学理通说采限制从属性说）。

**实践意义**：只有先确认正犯行为的存在与违法性，才能在此基础上判断共犯。

### 3.2 共同犯罪中的具体顺序

```
原则：贡献最大者/最接近结果者优先

具体操作：
1. 实行行为的直接实施者最先分析
2. 其次是分工中承担核心角色者
3. 再次是提供关键条件者
4. 最后是边缘参与者
```

### 3.3 间接正犯中的顺序

```
标准顺序：
1. 先分析直接行为者（被利用者）
   → 检验其是否成立正犯
   → 通常在某一阶层得出否定结论：
     - 缺乏故意（被欺骗）
     - 缺乏责任能力（未成年/精神病）
     - 受强制（缺乏期待可能性）
   
2. 再分析幕后操控者
   → 论证其间接正犯地位
   → 利用对方的"缺陷"实现犯罪
```

**例外**：当幕后者的正犯地位已经非常明显且不依赖于对直接行为者的分析时，可酌情调整。

### 3.4 被害人行为的分析

某些情况下需要分析被害人的行为：

- 被害人是否存在过错（关系到正当防卫的判断）
- 被害人的承诺是否有效（关系到违法性排除）
- 被害人的自我危殆（关系到客观归责）

## 四、罪名穷尽原则

### 4.1 原则内容

> **每一个在案件事实中具有合理适用可能性的罪名，都必须被检验，不得遗漏。**

### 4.2 "合理适用可能性"的判断

| 应当检验 | 无需检验 |
|----------|----------|
| 案件事实中的行为在外观上可能涵摄入该罪名 | 案件事实与该罪名明显无关 |
| 学说或实务中对此类行为存在该罪名的争议 | 仅因罪名中出现类似文字 |
| 司法实践中此类行为曾以该罪名追诉 | 纯粹理论假设无事实基础 |
| 题目暗示需要讨论（如涉及相关法条的引用） | — |

### 4.3 穷尽检验的实践边界

**穷尽不等于无限**：

- 对明显不成立的罪名，使用裁判式一句话排除即可
- 对争议性罪名，需要完整的鉴定式检验
- 对核心罪名，需要深入的涵摄分析

### 4.4 常见遗漏清单

以下罪名容易被遗漏，需要特别注意：

| 容易遗漏的罪名 | 典型遗漏场景 |
|---------------|-------------|
| 非法拘禁罪 | 绑架/抢劫过程中的限制自由 |
| 窝藏罪/掩饰隐瞒犯罪所得罪 | 帮助处理赃物 |
| 包庇罪 | 事后帮助逃避处罚 |
| 遗弃罪 | 重伤后弃之不顾 |
| 过失致人死亡罪 | 故意伤害致死不成立时的退路 |
| 强制猥亵罪 | 强奸未遂时的退路 |
| 寻衅滋事罪 | 公共场所的暴力行为 |

## 五、"结论一致则不问理由"原则（要点）

> **当不同学说、理论或解释路径对同一问题得出相同结论时，无需展开学说争议，直接得出结论即可。**

适用条件：（1）存在两个以上理论路径；（2）所有路径均指向相同结论；（3）结论本身可直接服务于涵摄。

注意事项：本原则仅适用于**最终结论相同**的情形；中间结论相同但可能影响后续路径的，仍需展开。应在脚注点明争议存在与各说结论一致的事实，避免读者误以为遗漏。

> **完整规则与示例**：参见 `methodology/proportionality.md §四.1 / §四.2`（结论一致跳过、不一致必须展开的细分情形与展开深度分级）。

## 六、检索顺序总结与操作指引

### 6.1 完整操作流程图

```
案件事实
  │
  ├─[第一步] 事实单元划分
  │   ├─ 单元A：甲对乙的暴力行为
  │   ├─ 单元B：甲取走丙的财物
  │   └─ 单元C：丁帮助甲逃离现场
  │
  ├─[第二步] 参与人列举
  │   ├─ 单元A：甲（直接行为人）、乙（被害人）
  │   ├─ 单元B：甲（直接行为人）
  │   └─ 单元C：甲（被帮助者）、丁（帮助者）
  │
  ├─[第三步] 逐人逐罪检验
  │   ├─ 单元A-甲：
  │   │   ├─ 故意杀人罪（重→先）→ 三阶层检验
  │   │   ├─ 故意伤害罪 → 三阶层检验
  │   │   └─ ……
  │   ├─ 单元B-甲：
  │   │   ├─ 抢劫罪（特别法→先）→ 三阶层检验
  │   │   ├─ 盗窃罪 → 三阶层检验
  │   │   └─ ……
  │   └─ 单元C-丁：
  │       ├─ 窝藏罪 → 三阶层检验
  │       └─ ……
  │
  ├─[第四步] 竞合处理
  │   ├─ 单元A内：故意伤害(+) 与 寻衅滋事(+) → 想象竞合
  │   ├─ 单元A与单元B之间：实质竞合
  │   └─ ……
  │
  └─[第五步] 全案结论
      ├─ 甲：……
      └─ 丁：……
```

### 6.2 检索顺序速查表

| 情境 | 检索顺序 |
|------|----------|
| 造成死亡结果 | 故意杀人→故意伤害致死→过失致人死亡 |
| 取得他人财物 | 抢劫→盗窃→诈骗→侵占→职务侵占 |
| 人身自由受限 | 绑架→非法拘禁→拐卖 |
| 暴力行为 | 故意杀人→故意伤害→寻衅滋事→聚众斗殴 |
| 涉及性行为 | 强奸→强制猥亵→猥亵儿童 |
| 涉及公共安全 | 以危险方法危害公共安全→交通肇事 |
| 涉及毒品 | 贩卖毒品→运输毒品→非法持有毒品 |
| 涉及伪造文书 | 伪造国家机关公文→伪造公司印章→伪造身份证件 |

### 6.3 注意事项

1. 检索顺序是**论证展示顺序**，不是思维发现顺序
2. 在实际检验中，可以先快速浏览所有可能罪名，再按顺序详细检验
3. 如果先检验的罪名已经成立，后续罪名仍需检验（穷尽原则），但竞合时可能被排除
4. 检索顺序的调整应在报告开头简要说明理由
